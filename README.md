# HY-Ego

> 从零开始构建数字生命 —— 工程测试版  
> Windows / Visual Studio 2022 / C++20 Modules（`.ixx`）

HY-Ego 是一个面向“数字生命（Digital Life）”的工程化试验仓库：以 **场景—存在—特征** 为基础信息骨架，向上构建 **世界树 / 概念树 / 二次特征 / 动态 / 因果 / 任务 / 需求 / 方法** 等结构；同时包含 **相机接入、三维场景管理与实时显示线程** 等工程能力，便于在真实数据流中持续迭代验证。

## ✅ 环境要求

- Windows 10/11
- Visual Studio 2022（建议安装 “使用 C++ 的桌面开发” 工作负载）
- 需要启用 C++20（本工程大量使用 `.ixx` 模块接口文件）

> 如果你启用“相机/3D 显示”相关功能，通常还需要安装并配置相应 SDK（例如 RealSense SDK、以及你工程中选择的 3D/图形/视觉依赖）。不同机器的依赖路径可能不同，建议按你本机开发环境为准在 VS 工程属性里配置。
