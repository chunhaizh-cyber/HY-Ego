# 关系仓库节点移动统一入口 Codex 断点清单

日期：2026-07-06
状态：已实施 / 已验证

## 1. 输入入口

用户要求不要在每个服务重复实现节点移动逻辑，按建议把通用移动能力收束到一个通用入口。

## 2. 本次口径

```text
节点移动 = 普通父子关系重挂。
通用入口 = 关系仓库::重挂节点(节点, 新父节点)。
各领域服务不得重复实现移动算法。
领域服务只做本领域前置拒绝，再调用通用入口。
```

## 3. 本次代码修改

```text
关系仓库::重挂节点 增加入参有效性之外的移动拒绝：
- 不能移动到自身下面
- 不能移动到自身子孙下面
- 不能移动到当前父节点导致无结构变化
- 没有当前父关系时不能伪装成移动成功
```

## 4. 当前验证点

```text
入口样例新增：
重挂拒绝: 自身=是 子孙=是 当前父=是
```
