## 当前现象

`x64/Debug/海中鱼巣.exe` 完成默认入口自检后立即退出，退出码为 0。

## 直接生成点

`海中鱼巣/入口.cpp` 在全部自检完成后直接执行 `return 程序通过 ? 0 : 1`。THREAD-S2 至 THREAD-S6 的线程对象只用于启动、停止和收口验收，均在 `main` 返回前停止并 `join`。

## 当前层级

```text
L0：程序启动后立即退出。
L1：main 自检完成后直接返回。
L2：缺少自检成功后的常驻生命周期入口。
L5：既有 THREAD-S2 至 THREAD-S6 只证明线程壳可启动和停止，不证明程序常驻。
```

## 选中分支

保留全部现有自检。自检通过后，默认进入低频休眠常驻循环；收到 `SIGINT`、`SIGTERM` 或 Windows `SIGBREAK` 时退出循环并正常返回 0。

增加外部验收参数：

```text
--self-test-exit
```

该参数只用于完成全部自检后返回，避免自动构建验收被默认常驻模式阻塞。

## 已确认事实

```text
默认运行在 750 毫秒后仍保持存活。

常驻循环不写仓库、领域结构或业务事实。
控制台与运行日志只做人读说明。
```
