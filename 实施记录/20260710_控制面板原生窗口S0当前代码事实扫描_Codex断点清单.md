## 6. 技术路线判断

第一轮推荐原生 Win32 窗口：

~~~text
复用当前 Windows C++ 工程。
不接 WebView、HTML、JSON 或浏览器服务器。
不接 SQL、ADO 或数据库运行期裁决。
不新增 UI 线程；自检通过后由 main 进入窗口消息循环。
窗口只读取控制面板服务材料和启动时结构统计快照。
默认运行打开窗口；--headless 保留当前常驻循环；--self-test-exit 保持不变。
~~~

## 7. 结构承载边界

计划新增的控制面板窗口材料只能是会话期人读 DTO：

~~~text
用途：为原生窗口提供只读总览、六类树根、结构计数和操作请求壳显示。
写入方：main 编排入口和控制面板服务只读入口。
读取方：控制面板窗口。
生命周期：当前进程和当前窗口会话；关闭窗口即销毁。
验证方式：窗口操作前后权威结构数量不变；DTO 不持久化，不回写领域结构。
~~~
