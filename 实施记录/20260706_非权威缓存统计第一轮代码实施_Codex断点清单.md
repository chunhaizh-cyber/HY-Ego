
已确认第一轮范围：

```text
定义缓存命名空间。
定义非权威结构统计快照。
实现结构统计运行期快照。
实现清空结构统计缓存。
建立清空全部缓存接口壳。
```

```text
主信息非权威缓存字段。
旧缓存字段迁移。
```

## 3. 核心处理

本切片新增 `统计服务`：

```text
定义缓存命名空间。
创建统计快照结构。
重建结构统计缓存。
读取结构统计缓存。
清空结构统计缓存。
清空全部缓存壳。
拒绝缓存裁决业务事实。
```

默认入口接入第一轮验证：

```text
空缓存读取可接受。
结构统计快照可由节点、关系、索引仓库当前公开读数重建。
清空结构统计缓存后，节点、关系、索引权威数量不变。
缓存不能作为业务事实裁决依据。
现有最小闭环仍通过。
```

## 4. 结构承载

代码修改：

```text
海中鱼巣/领域/统计服务.h
海中鱼巣/入口.cpp
```

结构映射：

```text
缓存命名空间：缓存类型、来源结构类型、来源版本、规则版本、缓存值类型、可持久化标记、失效条件。
结构统计快照：非权威运行期结构，保存节点数、关系数、索引数。
统计服务运行期缓存：只保存在服务对象内存中，不写节点、关系、索引或主信息。
```

## 6. 入口前置拒绝

本切片验证的拒绝路径：

```text
缓存类型未设置的命名空间被拒绝。
无效缓存命名空间不能清空结构统计缓存。
缓存命名空间不能裁决需求满足、任务完成、方法成功或世界事实。
清空缓存不删除、不修复、不补写节点、关系或索引。
```
