## 4. 工作线程所有权与停止语义

```text
任务工作线程::取出一条任务结果回执()
-> 内部锁内检查停止状态和队列
-> 移动队首值式回执并 pop_front
-> 返回后释放队列锁
-> 路由在锁外调用组合器
```

停止锁存后新回执发布被拒绝；停止前已入队回执仍可被排空，取出结果同时返回 `队列已停止`。#225 不需要修改任务工作线程、回执协议或 #224 的任何生产模块。
