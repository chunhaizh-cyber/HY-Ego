## 2. JY-379 唯一裁决

```text
第一层：因果完整结构键召回

当前桶项不同 -> 哈希碰撞，保留不同事实并继续扫描，不返回。

本层不形成 O1 身份或同账结论。

第二层：O1 身份
只比较完整任务句柄 + 非零任务序号。
任一字段不同 -> 不同 O1 身份 / 新账项，不进入该既有账项的核心或阶段比较。

第三层：O1 同账
前提是完整 O1 身份相同，且新候选与可选既有候选各自合法。
同核心 + 同阶段 -> 幂等。
同核心 + 待结算推进合法最终 -> 允许推进。
异核心或不允许阶段转换 -> 同账冲突，零结构变化。
候选自身非法或同一身份多个已发布当前记录 -> 追根因，不归同账冲突。
```
