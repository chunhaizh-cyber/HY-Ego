## 4. 结构承载

```text
方法身份：节点仓库中的 方法 节点类型。
方法首角色：方法节点到抽象状态节点的 模板 关系，顺序号为方法角色顺序号。
条件 / 结果节点：方法首到条件 / 结果方法节点的 模板 关系，并由角色状态区分。
条件 / 结果场景：条件 / 结果节点到场景节点的 引用 关系。
方法虚拟存在：方法首到存在节点的 引用 关系，要求唯一。
拒绝路径：错误身份、错误角色、错误场景读取和多虚拟存在读取均返回空，不写新节点、关系或索引。
```
