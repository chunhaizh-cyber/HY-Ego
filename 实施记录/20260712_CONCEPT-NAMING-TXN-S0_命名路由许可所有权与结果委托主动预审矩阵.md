## 7. JY-289 追加裁决

1. 不采用“许可前值式快照 + 许可内版本复核”。现有快照入口本身持活动图锁后进入接域仓库，无法保证结构许可先于活动图锁。
2. #213 允许定向修改 `概念图服务.h` 与 `语素服务.h`，只增加消费 `const 结构事务令牌&` 的只读重载和最小私有辅助；普通无令牌入口继续兼容 S1，禁止改写语义或扩为写入口。
3. S3 正式调用链固定为：`纯消息复核 -> 结构独占 -> 目标概念分片 -> 概念图令牌重载 / 活动图读锁 -> 生命周期关系令牌读取 -> 语素令牌重载 -> 关系来源与节点类型令牌读取 -> 需求服务令牌路径`。
4. 当前请求消失、版本漂移或名称出现均在任何需求写入前逻辑内返回；活动快照与生命周期内部矛盾继续追根因，不得伪装成消息过期。
5. 本追加裁决只修订 #213，不改变 #214 共享许可或 #215 委托 #225 的既定边界。
