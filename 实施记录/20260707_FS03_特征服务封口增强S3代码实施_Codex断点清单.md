## 2. 核心处理

```text
在 特征服务 中新增 写入宿主I64特征状态材料、读取宿主I64特征状态材料。
宿主必须已有实例特征槽位。
槽位下无特征值时，经特征服务内部既有 创建特征值 创建归属特征值节点。
I64 写入和读取继续复用既有 特征值服务 入口，但只通过特征服务对外暴露。
非归属特征值写入和读取保持拒绝，原值不变。
```

## 3. 结构承载

```text
输入入口：宿主节点、特征类型节点、I64 材料值。
核心处理：先找宿主槽位，再找或创建槽位归属特征值，最后写入 I64 材料。
结构承载：特征槽位到特征值节点的归属关系；特征值主信息 I64 材料沿既有服务承载。
输出变化：默认入口新增 FS-03 S3 I64 写入读回、FS-03 S3 非归属写入拒绝、FS-03 S3 非归属读取拒绝。
调用路径：入口.cpp -> 特征服务 -> 特征值服务 / 关系仓库 / 主信息仓库。
```
