## 2. 核心处理

已新增 `海中鱼巣/核心/运行宿主.h`，并在 `海中鱼巣/入口.cpp` 接入 THREAD-S2 默认入口自检。

本切片只实现运行宿主工程生命周期壳：

```text
运行宿主状态。
启动入口。
重复启动拒绝。
停止请求入口。
停止消息入队。
收口等待入口。
调度周期消息材料。
```

## 3. 结构承载

```text
运行宿主状态：`运行宿主状态` 局部枚举。
局部失败材料：`运行宿主操作结果`。
线程生命周期：`运行宿主` 内部 `std::thread`。
调度材料：`运行消息类型::调度周期` 写入 `有界运行消息队列`。
停止材料：`运行消息类型::停止` 以最高优先级写入 `有界运行消息队列`。
```

上述结构只承载工程生命周期和调度材料，不写节点、主信息、关系、索引、需求、任务、方法、状态、动态或因果引用。
