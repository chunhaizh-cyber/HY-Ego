# 需求目标状态承载专项 Codex 断点清单

日期：2026-07-05
状态：DV-1 已验证

## 1. 输入入口

用户确认：

```text
需求的是状态，不是 I64。
特征值只有特征类能够直接访问。
高级服务只允许直接用特征、存在、二次特征、状态、动态、因果服务。
场景服务也可以用。
```

本切片按该口径修正此前“需求目标值承载专项”的错误表述和代码路径。

## 2. 本切片目标

```text
需求目标改为目标状态承载。
I64 只作为第一轮状态值材料或特征状态材料。
需求服务不得写需求节点主信息值容器保存目标状态。
需求服务和方法服务不得直接依赖特征值服务。
特征值服务只允许由特征服务直接访问。
```
