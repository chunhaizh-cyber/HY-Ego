## 4. 核心处理

新增 `缓存事件线程壳`：

```text
缓存统计线程壳。
事件日志线程壳。
缓存刷新队列。
事件日志队列。
缓存刷新消息发送、满载局部失败和消费材料。
事件日志消息发送、模拟写入失败和消费材料。
停止消息、停止态和 join 收口。
```

本线程壳只搬运运行消息和产生人读 / 复核材料：

```text
写业务事实 = false
裸写仓库 = false
裁决业务事实 = false
修复结构 = false
恢复运行期结构 = false
```

默认入口新增 THREAD-S4-A1 到 THREAD-S4-A6 自检：

```text
THREAD-S4-A1 缓存统计线程可启动和停止。
THREAD-S4-A2 缓存刷新消息局部拒绝。
THREAD-S4-A3 清空缓存不改变权威结构。
THREAD-S4-A4 事件日志线程可启动和停止。
THREAD-S4-A5 事件日志失败不裁决机器事实。
THREAD-S4-A6 禁止 SQL / ADO / WebView / 外设接入。
```

## 5. 结构承载

```text
运行消息队列：只承载缓存刷新消息、事件日志消息和停止消息。
运行宿主.h：承载缓存 / 事件线程生命周期壳与局部失败材料。
统计服务：沿用既有非权威结构统计缓存入口，验证清空缓存不改变权威结构。
日志系统：沿用事件日志不裁决运行期事实、不直接恢复运行期结构边界。
入口.cpp：承载本切片默认入口验收样例。
```

未新增业务状态节点、主信息字段、关系、索引或旧项目字段。
