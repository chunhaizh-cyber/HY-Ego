# 流程图免确认直接生成详细设计口径修订 Codex 断点清单

日期：2026-07-08
状态：文档治理 / 规则口径修订 / 未入队 / 未改 C++ / 未构建 / 未迁移旧函数

## 1. 输入入口

```text
用户指令：流程图不在需要确认,将后续生成的流程图全部生成详细设计。
当前窗口：设计确认窗口。
```

## 2. 核心处理

```text
取消流程图到详细设计之间的单独流程图确认门禁。
后续流程图生成后，设计确认窗口可直接生成对应详细设计。
流程图仍只作为详细设计依据，不构成代码实施许可。
详细设计确认、施工计划确认、代码实施许可和可执行队列登记门禁保持不变。
本次同步 AGENTS、规则总纲、规则迁移清单、计划索引、当前状态、决策记录和待确认问题。
```


## 4. 输出变化

```text
后续流程图生成后不再新增单独“流程图是否确认”待确认项。
后续流程图生成后直接进入对应详细设计生成。
详细设计仍以已生成 / 待确认 / 未入队 / 未改代码 / 未构建 / 未迁移旧函数作为默认状态。
涉及施工计划、代码实施、旧能力迁移、字段落地或函数迁移时仍需另行确认和登记可执行队列。
```

```text

```
