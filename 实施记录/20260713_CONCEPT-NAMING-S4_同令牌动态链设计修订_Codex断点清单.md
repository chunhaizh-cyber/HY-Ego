## 3. 同令牌调用链

```text
统一命名路由取得共享许可
-> 任务服务同令牌重读选择组合身份
-> 方法服务同令牌重读方法投影、动作入口归属、当前状态与规格
-> 语素服务同令牌绑定并双向读回名称关系
-> 方法服务同令牌复核动作入口并委托动态服务
-> 动态服务同令牌创建实例动态和来源动作关系
-> 方法服务同令牌创建来源方法关系
-> 动态服务同令牌读回材料并复核完整性
-> 方法服务同令牌组合动作动态证据
-> 释放共享许可后生成值式回执
```

## 4. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 令牌无效、未接域或普通材料漂移 | 逻辑内返回 | 第一笔名称或动态写入前拒绝，结构不变化 |
| 路由持有效共享许可但下层转入无令牌入口 | 追根因解决 | 停止本次路径，不用兼容入口兜底 |
| 名称关系写后双向读回不一致 | 追根因解决 | 不进入动作动态和回执 |
| 名称完整但动态写入或读回不一致 | 追根因解决 | 保留有效名称事实，释放许可，不生成回执或结算 |
