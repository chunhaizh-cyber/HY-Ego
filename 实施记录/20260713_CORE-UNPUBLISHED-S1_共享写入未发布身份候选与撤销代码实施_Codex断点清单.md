
`主信息未发布候选` 与 `节点未发布候选` 均不可默认构造、不可复制、可移动但不可移动赋值；移动构造显式清空源候选。候选私有绑定创建仓库、完整值式令牌、精确句柄和局部阶段，析构不写结构。

## 3. 核心处理与结构承载

候选创建入口先验证共享路径令牌、节点类型和主信息身份，再由对应仓库建立有效记录并返回不可伪造候选。确认入口复核同仓库、同完整令牌、未发布阶段和精确有效记录，只把调用栈候选改为已确认，不修改仓库记录。

撤销入口只接受候选对象，不接受裸句柄；复核通过后在仓库本地独占锁内物理移除本候选记录。重复撤销返回 `幂等已撤销`，已确认候选、错仓库、错令牌、错线程和过期令牌均拒绝，精确记录漂移返回 `内部不一致`。

现有 `删除主信息(句柄, 令牌)` 与 `删除节点(句柄, 令牌)` 未放宽，仍调用 `验证独占令牌`。`结构事务接线.数据.h`、`协调.结构事务.ixx`、关系仓库、索引仓库和全部领域 / 线程文件零修改。

## 4. 输出变化与调用路径

```text
领域组合器持同一共享路径令牌
-> 创建主信息未发布候选
-> 以候选主信息创建节点未发布候选
-> 创建并读回本次非发布关系
-> 最终发布成功后显式确认节点和主信息候选

发布前失败
-> 先精确删除本次关系
-> 撤销节点候选
-> 撤销主信息候选
-> 物理记录与公开读取回到前态
```

独立真模块 `海中鱼巣.核心.自检.未发布身份候选` 承载验收正文；`入口.cpp` 只 import 模块并在最终回归阶段登记顺序 585。
