## 4. 本轮初步结论

```text
1. 旧语素系统已经把“词面 -> 语素入口 -> 基础信息节点”作为桥接主线，这个方向可保留，但新项目不得复制旧语素树和旧主信息字段。
2. 旧 `对应基础信息`、`对应基础信息类型`、词性默认映射和信息入口定义需要改写为语素服务、关系仓库、索引仓库和候选复判规则。
3. SQL 投影、显示标题、语言命名和词性文本映射只做人读或后续显示 / 持久化适配，不得作为机器事实。
4. 旧自动“确保语素入口对应基础信息”会越过新项目领域服务边界，应拆成目标信息节点创建和语素绑定两个步骤。
5. FS-02 第一批只证明旧语素入口和基础信息桥接函数族已登记，不证明旧语素类已迁移，也不证明候选查询、自然语言理解或显示层已实现。
```
