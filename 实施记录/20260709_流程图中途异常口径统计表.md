# 流程图中途异常口径统计表

日期：2026-07-09

## 1. 统计口径

本表按用户补充口径统计流程图中的“异常”：

```text
异常 = 流程已经进入某个中途阶段后，在某种条件下返回非预期结果。
```

不计为异常：

```text
写入前的入口拒绝。
只读候选为空。
非权威缓存缺失。
显示 / 日志 / 控制台 / 回执的人读材料。
```

计为中途异常：

```text
前置拒绝已经通过后创建节点失败。
前置拒绝已经通过后写主信息、关系、索引、状态、动态或结算记录失败。
已经创建部分结构后返回空句柄、无效句柄、false 或失败回执。
写入后读回不符合预期。
多步写入未证明事务回滚、失效隔离或数量快照级半结构不可读。
中途返回“不能声明成功”但没有标明逻辑错误、事务缺口或停止依赖路径。
```
