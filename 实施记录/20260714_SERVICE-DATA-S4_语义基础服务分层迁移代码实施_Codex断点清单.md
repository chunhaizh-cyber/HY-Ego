## 4. 准入、失败收口与并发

主键为零、句柄无效、文本含空白 / 分隔符 / 超长、关系角色非法、目标不在对应白名单和错误入口类型均在第一笔写入前拒绝。共享读后进入独占会话时重新复核目标节点类型和主信息；写前漂移返回版本漂移。

主信息候选、节点候选、主键和关系在一次会话内按顺序写入，任一步失败或异常均由执行器在许可释放前逆序撤销。故障自检分别命中主信息、节点、主键、对应关系和追溯关系五个写点，节点 / 关系 / 索引数量及所有捕获句柄均回前态。

基础信息、语素入口、语素概念入口、已有语素关系追加和轻量因果五类同事实六线程竞争均只形成一次提交，其余为同事实幂等或明确许可拒绝；最终权威读回完整，无重复关系。
