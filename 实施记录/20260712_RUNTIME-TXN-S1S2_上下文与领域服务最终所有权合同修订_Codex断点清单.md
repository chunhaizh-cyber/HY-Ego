## 4. 最终对象与模块方向

### 4.1 对象所有权

```text
运行期上下文
  结构事务协调器
  结构事务接线
  主信息仓库
  节点仓库
  关系仓库
  索引仓库
  领域服务装配

运行期上下文宿主
  发布互斥 / 一次状态
  已发布上下文
  租约源
```

宿主与上下文位于同一 `启动.运行期上下文` 模块，但宿主不是上下文成员；消费者停止并释放租约后，宿主才释放上下文。

### 4.2 模块依赖

```text
启动.运行期上下文 import 装配.领域服务
装配.领域服务 -> 只接收接线与四仓库引用
装配.领域服务 -X-> 不 import 启动.运行期上下文
```

服务装配模块不拥有仓库、宿主、租约或协调器。禁止无类型槽、`void*` 服务所有权、临时假服务、循环 import 和第二服务宿主。

## 5. 构造与析构

成员声明 / 构造顺序：

```text
协调器 -> 接线 -> 主信息 -> 节点 -> 关系 -> 索引 -> 领域服务装配
```

析构顺序自动逆向：

```text
领域服务装配 -> 索引 -> 关系 -> 节点 -> 主信息 -> 接线 -> 协调器
```

该顺序兼容节点依赖主信息、关系 / 索引依赖节点。上下文仍有活动租约时不得析构。
