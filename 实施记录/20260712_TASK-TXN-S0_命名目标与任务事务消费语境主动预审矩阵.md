## 3. 假定接口契约

```text
传统服务：只 include 结构事务接线.数据.h，保存可空接线，接收并验证 const 值式令牌。
隔离自检：import 协调.结构事务，拥有移动许可，调用传统服务令牌重载。
生产真模块：在调用期 import 协调模块、取得许可、下传令牌；线程 / 消息 / 回执不保存许可。
兼容路径：保留既有调用所需的无接线入口，但不参加结构事务完成声明。
运行期主装配：不在 #221-#225 临时接线，等待 #246 扫描和后继计划。
```
