## 3. 正式裁决

| 编号 | 裁决 |
| --- | --- |
| JY250-D1 | 第一版授权材料必须同时返回两条承接关系完整句柄 / 版本，并证明同一任务和需求端点。 |
| JY250-D2 | `任务方法选择请求` 与当前选择材料固定由 `任务服务.h` 定义；方法召回服务单向依赖任务服务并适配完整建议，方法服务单向依赖任务服务并形成精确动作桥；任务服务不包含任一方法侧头文件或解释方法召回 DTO。 |
| JY250-D3 | “扁平”不等于裁剪：请求必须冻结批次身份、索引发布版本、方法登记 / 生命周期 / 结果规格 / 条件 / 上下文 / 动作来源句柄与版本。 |
| JY250-D4 | 第一版只接受来源当前的 `唯一可建议`；无候选、过期、语义并列和无独立权威来源的 `策略唯一` 均不提交。 |
| JY250-D5 | 当前选择由唯一有效顺序号 30 `引用(任务 -> 方法)` 关系完整句柄 / 版本和所属筹办生命周期版本共同标识；顺序号只标识角色槽位。 |
| JY250-D6 | 精确同请求幂等；同一生命周期版本异方法拒绝；只有 #222 正式返回新筹办中生命周期后才失效旧选择、保留审计并创建新选择。 |
| JY250-D7 | 动作桥只接受任务服务读出的精确当前选择材料；任意任务到方法引用、失效选择和旧生命周期选择均不构成执行准入。 |
| JY250-D8 | #214 和 #224 必须消费双向授权、生命周期和精确当前选择关系 / 版本，不得自行从任意引用重建选择。 |

## 4. 假定正式接口

```text
任务服务::读取任务承接授权材料(任务)
方法召回服务::适配任务方法选择请求(完整方法选择建议, 任务承接授权材料)
任务服务::提交任务方法选择(任务方法选择请求, 同域令牌)
任务服务::读取当前任务方法选择材料(任务)
任务服务::复核当前任务方法选择仍为当前(当前选择材料)
任务服务::读取任务方法选择审计(任务)
方法服务::读取动作执行桥请求材料(当前任务方法选择材料)
```

以上是依赖门控假定契约，不是当前实现事实。#223 执行前必须读取 #221 / #222 实际接口逐项映射；漂移时不改代码并退回设计。

## 5. 非成功二分

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 无候选、过期、语义并列、无来源策略唯一 | 逻辑内返回 | 保持筹办中，不写选择 |
| 双向授权、生命周期或任一来源正常漂移 | 逻辑内返回 | 本次请求不写，既有合法选择保持 |
| 精确重复 | 幂等逻辑内返回 | 读回既有选择，不增长结构 |
| 同生命周期版本异方法 | 逻辑内返回 | 不覆盖当前选择 |
| 任意引用或旧选择进入动作桥 | 逻辑内返回 | 不形成执行材料 |
| 当前选择多条、端点矛盾、写后唯一读回失败 | 追根因解决 | 停止选择和执行依赖，不扫描挑选或修复 |
