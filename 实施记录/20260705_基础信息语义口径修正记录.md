# 基础信息语义口径修正记录

日期：2026-07-05

## 1. 用户口径

用户确认：

```text
基础信息是指 特征,存在,场景,状态,动态,二次特征,因果等信息
```

## 2. 修正结论

本项目采纳该口径：

```text
基础信息 = 基础信息类信息节点集合
```

当前基础信息类信息节点包括：

```text
基础信息通用节点
存在
场景
特征
状态
动态
二次特征
因果引用
```

`基础信息` 不再被解释为只等于 `节点类型::基础信息`。

## 3. 代码影响

本轮调整：

```text
新增 `节点类型::二次特征`。
新增 `特征服务::创建二次特征`。
语素服务的可绑定信息节点范围扩大为基础信息类节点、需求、任务、方法。
语素入口自检改为允许存在、场景、特征、状态、动态、二次特征、因果引用绑定。
特征值节点第一轮仍不默认允许作为语素入口对应信息节点。
```
