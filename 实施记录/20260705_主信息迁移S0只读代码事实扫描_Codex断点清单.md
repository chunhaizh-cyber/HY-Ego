## 6. 扫描事实

### 6.1 主信息仓库

已存在结构：

```text
主信息记录
主信息编号
版本号
状态
拓扑锚点编号
std::vector<std::int64_t> 值容器
```

已存在函数：

```text
主信息仓库::创建主信息()
主信息仓库::读取主信息(主信息句柄)
主信息仓库::删除主信息(主信息句柄)
主信息仓库::主信息是否有效(主信息句柄)
主信息仓库::写入I64值(主信息句柄, std::int64_t)
主信息仓库::写入I64值(主信息句柄, std::uint64_t 值索引, std::int64_t)
主信息仓库::读取I64值(主信息句柄)
主信息仓库::读取I64值(主信息句柄, std::uint64_t 值索引)
```

入口拒绝事实：

```text
读取主信息会拒绝仓库编号不匹配、版本号不匹配、已删除记录。
删除主信息会拒绝不存在、仓库不匹配、版本不匹配、已删除记录。
写入 I64 值会拒绝不存在、仓库不匹配、版本不匹配、已删除记录和非法值索引。
读取指定索引 I64 值在未写入该索引时返回空。
```

未发现：

```text
VecI64 / VecIU64 写入函数。
拓扑锚点绑定函数。
非权威缓存登记函数。
旧项目各类 *主信息记录。
```

### 6.2 特征值服务

当前 `特征值服务` 事实：

```text
创建特征值() 创建 节点类型::特征值。
写入I64值(特征值节点, 值) 先读取节点，要求节点类型为 特征值，再写入该节点主信息的 I64 值。
读取I64值(特征值节点) 先读取节点，要求节点类型为 特征值，再读取该节点主信息的 I64 值。
```

判断：

```text
特征值服务当前仍是特征值节点封口入口。
当前未提供按值索引写入 / 读取特征值的领域入口。
```

### 6.3 语素服务和关系仓库

当前 `语素服务` 事实：

```text
创建语素入口(对应信息节点, 主键) 会先检查对应信息节点是否可绑定。
可绑定类型包括基础信息、存在、场景、特征、非运行期临时状态、非运行期临时动态、二次特征、因果引用、需求、任务、方法。
创建语素入口时会创建 关系类型::语素对应信息。
读取入口绑定目标(语素入口) 返回 std::optional<节点句柄>。
创建概念入口(...) 会创建 关系类型::语素概念追溯。
读取入口概念追溯(语素入口) 返回 std::optional<节点句柄>。
```

当前 `关系仓库` 事实：

```text
关系类型::语素对应信息 和 关系类型::语素概念追溯 被 关系需要唯一目标(...) 判定为源节点唯一目标关系。
创建关系时，如果同一源节点已存在同类型 语素对应信息 或 语素概念追溯 关系，会拒绝新增。
```

S0 缺口：

```text
已确认口径要求语素入口可对应多个对应信息节点，概念追溯可多级、多路。
当前代码只能读取单个绑定目标和单个概念追溯目标。
当前关系仓库还会拒绝同一语素入口的多个语素对应信息 / 语素概念追溯目标。
```

### 6.4 需求 / 方法服务与主信息值容器

当前 `需求服务` 事实：

```text
创建I64目标需求(std::int64_t 目标值) 会创建需求节点主信息，并直接把目标值写入该主信息 I64 值容器。
结算I64目标(...) 会从需求节点主信息读取目标值，再读取特征值服务当前值。
```

S0 缺口：

```text
已确认口径要求 MI-V1 值容器语义只保留在特征值系统内，不扩展为需求 / 任务 / 方法通用值容器。
当前需求服务仍直接使用需求节点主信息值容器承载 I64 目标值。
这不是本切片新增问题，但会阻断后续主信息迁移收口声明。
```

当前 `方法服务` 事实：

```text
执行I64写入方法(...) 要求任务节点有效、方法节点有效、特征值节点有效，并要求存在 任务节点 -> 方法节点 的引用关系。
执行写值时通过 特征值服务::写入I64值 写入目标特征值节点。
```

判断：

```text
方法服务当前写值路径经特征值服务封口，不直接写主信息值容器。
```

## 7. S0 结论

已确认事实：

```text
主信息仓库最小函数已存在。
单槽和分槽 I64 值容器函数已存在。
旧项目各类 *主信息类未出现在当前新项目代码中。
特征值服务当前通过特征值节点封口使用主信息 I64 值。
方法服务执行写值当前经特征值服务封口。
```

阻断后续主信息迁移收口的缺口：

```text
缺口 1：语素对应信息和概念追溯当前仍是单目标接口和单目标关系，不符合已确认的多对应、多级、多路口径。
缺口 2：需求服务当前直接把 I64 目标值写入需求节点主信息值容器，不符合 MI-V1 只保留在特征值系统范围内的确认口径。
缺口 3：说明书/函数功能说明文档.md 不存在，后续函数迁移前缺函数功能说明依据。
```
