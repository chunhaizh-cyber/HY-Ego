## 3. 结构承载

新增结构：

```text
海中鱼巣/核心/运行消息队列.h
```

结构说明：

| 结构 | 用途 | 权威性 |
| --- | --- | --- |
| `运行线程角色` | 标识消息来源或目标角色 | 非事实 |
| `运行消息类型` | 区分调度、停止、任务请求、回执、缓存、事件、外部材料等消息 | 非事实 |
| `运行消息优先级` | 承载停止消息最高优先级和普通排序材料 | 非事实 |
| `运行消息` | 承载请求、材料、回执、调度信号和复核材料 | 非事实 |
| `有界运行消息队列` | 固定容量、局部失败、停止收口和基础 push / pop 壳 | 工程承载，不裁决机器事实 |
