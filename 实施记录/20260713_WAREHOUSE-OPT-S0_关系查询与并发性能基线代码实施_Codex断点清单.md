## 2. 隔离夹具与测量合同

新增真模块 `海中鱼巣/核心/自检.关系仓库性能基线.ixx`，固定参数如下：

```text

局部节点：1,024
预热：每查询每轮 20 个批次
正式样本：每查询连续三轮合计 600 个批次
批内重复：1k / 10k / 100k 分别为 64 / 8 / 1
写入样本：创建 / 失效 / 重挂 / 删除各 200
并发：4 线程、固定 10 秒、约 90% 读 / 10% 重挂写
稳定门槛：(三轮最大中位数 - 最小中位数) / 最小中位数 <= 15%
```

夹具覆盖正向、反向、存在、节点相关和当前审计的稀疏 / 密集查询，以及多根、多父共享节点、概念图环和普通父子单父拒绝。独立参考表逐项复核关系编号、类型、完整端点、顺序号、版本和状态。

等待代理值由查询耗时相对单线程参考值形成，只用于同机前后对比，不是 `shared_mutex` 内部精确等待时间。
