## 4. 当前判断

1. 旧哈希、命中次数、最后命中只能作为非权威缓存方向，不能迁入主信息权威字段。
2. 旧需求累计结算和最近结算字段只能作为需求结算摘要缓存方向，权威结算必须读取需求结算记录。
3. 旧结构统计可以作为第一轮统计快照输入，但不能接 SQL 快照裁决运行期事实。
4. 旧观测缓存、二次特征签名缓存和日志耗时统计均不得作为业务事实。
5. 控制面板、SQL、D455、体素和外设统计继续归入 FS-10 或 FS-X 后续专项。
