## 3. 核心处理

本切片改造 `方法服务`：

```text
新增动作入口类型枚举。
新增动作入口状态枚举。
新增动作入口角色状态。
新增登记动作入口入口。
新增读取方法动作入口入口。
新增方法是否具备主动动作入口入口。
新增绑定和读取动作输入规格入口。
新增绑定和读取动作输出规格入口。
新增记录动作动态证据入口。
方法执行特征状态写入前增加主动动作入口前置拒绝。
```

同时补齐 `关系仓库` 按顺序号读取目标节点组的只读入口，用于区分方法角色状态、动作入口状态、动作输入规格和动作输出规格。

## 4. 结构承载

代码修改：

```text
海中鱼巣/领域/方法服务.h
海中鱼巣/核心/关系仓库.h
海中鱼巣/核心/关系仓库.cpp
海中鱼巣/入口.cpp
```

结构映射：

```text
动作入口角色节点：节点类型::方法 + 方法角色状态::动作入口。
动作入口当前状态：抽象状态节点 + 动作入口状态顺序号模板关系。
方法动作入口关系：关系类型::引用，方法首节点 -> 动作入口角色节点。
稳定动作键：索引仓库主键 -> 动作入口角色节点。
动作输入规格：关系类型::引用，动作入口角色节点 -> 场景节点，输入规格顺序号。
动作输出规格：关系类型::引用，动作入口角色节点 -> 场景节点，输出规格顺序号。
动作动态来源：动态服务 来源动作 指向动作入口角色节点；动态节点另有来源方法引用关系。
```

## 6. 入口前置拒绝

本切片验证的拒绝路径：

```text
无效方法首登记动作入口被拒绝。
稳定动作键为 0 被拒绝。
同一方法重复登记当前动作入口被拒绝。
不同方法复用同一稳定动作键被拒绝。
无动作入口的方法执行特征状态写入被拒绝。
缺任务方法关系的方法执行仍被拒绝。
拒绝前后节点、关系和索引数量不变。
```

## 7. 调用路径

```text

-> 方法服务登记动作入口
-> 状态服务创建动作入口角色状态和入口状态
-> 关系仓库写方法首到动作入口角色引用关系
-> 索引仓库绑定稳定动作键
-> 方法服务绑定输入规格和输出规格场景
-> 方法执行入口检查任务方法关系和主动动作入口
-> 方法服务记录动作动态证据
-> 动态服务记录实例动态，来源动作指向动作入口角色节点
-> 关系仓库补充动态到来源方法引用关系
```
