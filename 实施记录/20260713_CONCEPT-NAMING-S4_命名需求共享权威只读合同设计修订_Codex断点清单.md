## 3. 裁决

采用需求服务共享路径权威只读 DTO，不新增不可伪造授权能力。公开入口固定为语义：

```text
读取概念命名需求权威材料(需求节点, const 结构事务令牌&)
```

入口只接收需求句柄和令牌，不接收预期目标概念、概念命名需求参数、绑定候选或任务材料。需求服务必须自行从仓库派生：

```text
读取状态：已拒绝 / 可绑定 / 已满足 / 内部不一致
需求、主体、目标宿主、场景
目标状态、目标状态关系
目标概念、目标概念关系
服务根需求、服务根发布关系
```

名称绑定路由必须比较候选需求、来源任务承接需求和 DTO 需求，并比较消息 / 当前待命名请求 / 候选目标概念与 DTO 目标概念；后续名称关系、动态目标和回执只使用 DTO 目标概念。
