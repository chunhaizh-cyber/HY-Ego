## 4. 目标服务和目标结构

| 项 | 目标口径 |
| --- | --- |
| 方法服务 | 继续作为方法身份、方法首、条件 / 结果、候选材料读取和方法关系读取入口 |
| 任务服务 | 只提供已确认的任务承接、任务选择方法和实际结果状态材料 |
| 特征服务 / 状态服务 | 后续条件材料必须经已确认服务入口读取，不直接访问特征值服务 |
| 关系仓库 | 承载方法首、条件 / 结果、父 / 前置 / 后续方法和来源任务引用关系 |
| 索引仓库 | 只做候选召回入口，不裁决方法可执行事实 |
