## 2. 预冻结待实现接口合同

唯一生产合同为 P3 v0.5 子计划第 2 节 `NT-C3/v0.5`；本任务只实现 `C3-E1—C3-E3`，消费 `C3-A1—C3-D2` 和 `C3-A0`。对每个具名请求类型分别提供合同第 2.5 节规定的强类型重载集合；不得保留通用请求体、`variant`、字节载荷或兼容重载。M01—M03、M26 只在切换库存字符串中引用 `NT-C2/v0.4` 的真实 / 预冻结签名，不因此取得 P2 生产 import 或写权。本文不复制 DTO、签名或 ABI 定义。

所有调用只使用稳定值类型和结构化结果，保持同键同义、冲突、许可拒绝、版本漂移、逻辑内返回和内部逻辑错误可区分。

本任务向 #348 / #352 提供：


提供者最终接口漂移时退回精确提供计划和本任务；本任务不得发明兼容重载、旧域回退或双写。
