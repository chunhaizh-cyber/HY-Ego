## 11. 入口拒绝与失败收口

| 场景 | 分类 | 动作 |
| --- | --- | --- |
| #242 未完成或实际接口与假定契约不一致 | 逻辑内返回 | 不改代码，退回设计计划窗口修订 |
| 当前代码锚点、验收编号集合或 300/310 材料边界漂移 | 逻辑内返回 | 更新事实和计划，不机械搬移旧行号 |
| 自检上下文缺少当前活动图或只读支持材料 | 逻辑内返回 | 不登记对应单元，不进入候选计算 |
| 运行器拒绝重复顺序 / 编号或空回调 | 逻辑内返回 | 不执行批次，不触碰业务结构 |
| 300 失败且当前活动图交接槽未封存完整材料 | 设计内自检合同失败 | 310 仍执行一次，只返回前置材料不完整，不运行候选场景 |
| 单项按既有语义不通过 | 设计内结果 | 保留原编号，继续其它单元，批次最终非零 |
| 迁移后 18 项编号缺失、重复或结果与基线不一致 | 追根因解决 | 停止 #217 解锁，定位首次偏差 |
| 310 在 300 前执行或读取非当前活动图材料 | 追根因解决 | 修正显式依赖和顺序 |
| 候选路径提交删除、重挂关系或改变活动结构 | 追根因解决 | 立即停止依赖路径，追查越权写入 |
