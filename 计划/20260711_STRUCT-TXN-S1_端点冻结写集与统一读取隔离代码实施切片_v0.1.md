## 5. 已验证当前事实

1. 四仓库各自持锁，节点有效性复核和关系 / 索引仓库写锁之间存在 TOCTOU。
2. 写入事务当前只有状态壳，无写集、隔离、回滚、补偿或领域服务接线。
3. 活动快照只隔离概念图服务读取，直接仓库和其它服务读取可见分步写。
4. 当前大量隔离自检独立构造仓库，不能强制它们虚假加入生产事务域。
5. 正式基线没有结构事务三文件；执行窗口仅建立了未提交骨架并完成首次 0/0 编译。骨架数据头只有裸指针和三个令牌字段，不含正式 ABI，不能作为实现事实继续扩散。
6. 四仓库为传统 `.h/.cpp`，不能直接持有命名模块归属类型；当前构造器均无事务接线，大量隔离夹具依赖旧签名。
7. 关系 / 索引多处持本仓库锁时读取节点；必须先复制并释放本仓库锁，再在同一结构许可期读节点。
8. `e21affd` 已形成最终回归 300 / 310 / 400；顺序 500 仍须执行前确认。
9. `写入事务` 只有状态与失败文本，不是 #217 的许可、令牌或回滚实现。

## 6. 假定接口契约

### 6.1 结构事务合同

```text
结构事务接线.数据.h：传统头可见的许可类型、域/运行期纪元/许可序号/类型令牌、`shared_ptr<void>` 运行期状态、移动 RAII 许可、释放函数和四函数接线表；无独立 .cpp。
协调.结构事务：非零域与纪元、共享/独占 RAII、许可序号、活动许可表、取得线程、重入拒绝、隐藏锁状态和接线工厂。
四仓库构造器尾部可选接入值式接线；未接域保持既有能力但不能参与安全删除。
无上游令牌的公开入口自行取得共享许可；嵌套调用显式传递同域令牌，下层只验证令牌且不重入共享许可。
独占令牌只可调用同域事务包入口。
节点删除和主信息删除必须取得独占许可；其它普通读取、新增、关系变化和索引变化取得共享许可。
锁序固定为 结构事务许可 -> 活动图锁_ -> 图写锁_ -> 单个仓库锁。
关系/索引锁内只复制值式候选，释放本仓库锁后以同一令牌读取节点。
共享运行期状态只作寿命桥；协调器对象、仓库和活动许可均共享该状态，最后一份引用释放后析构，不存在裸回调悬空；不持久化、不作为机器身份。
令牌验证同时匹配运行期状态、域、纪元、序号、许可类型、活动表和当前线程；字段复制、跨线程使用或过期令牌均不能通过。
同线程同状态无令牌重入属于实现错误并触发诊断；嵌套必须下传原令牌。
```

协调器许可只控制并发可见性，不承载业务事实。本切片不宣称回滚、补偿、崩溃恢复或完整 ACID。

## 11. 入口拒绝与失败收口

| 场景 | 分类 | 动作 |
| --- | --- | --- |
| `e21affd` 阶段总成 / 模块 / 仓库接口漂移或最终回归阶段顺序 500 已占用 | 逻辑内返回 | 不改代码、不自行改号，退回设计计划修订 |
| 域编号为零、参与者域不一致、令牌来自其它域、独占令牌调用普通共享路径 | 逻辑内返回 | 写前拒绝，不改变结构 |
| 未接域仓库调用既有单仓库入口 | 设计内允许分支 | 保留既有能力，但不得参与安全删除 |
| 取得独占许可前取消或候选过期 | 逻辑内返回 | 只返回拒绝材料，不写结构 |
| 运行器登记顺序冲突、上下文缺项或回调为空 | 逻辑内返回 | 不执行结构事务自检，不触碰业务结构 |
| 数据桥半接线、域为零但回调非空或协调器生命周期不足 | 逻辑内返回 | 写前拒绝；正式工厂产生该结果则追根因 |
| 令牌身份、锁序、嵌套许可或仓库身份在入口前置后仍不一致 | 追根因解决 | 停止依赖路径并定位首次偏差 |
| 独占持有者重入共享许可、出现 ABBA 或读者看见中间状态 | 追根因解决 | 不用超时 / 日志绕过 |
| A01-A14 缺失、重复、正文落回入口或自检进入生产依赖 | 追根因解决 | 不解锁 #218，修正承载边界 |
