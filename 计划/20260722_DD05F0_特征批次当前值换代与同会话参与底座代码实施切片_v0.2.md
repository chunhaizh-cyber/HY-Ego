## 5. 固定机器和接口合同

### 5.1 核心多参与者执行

`结构写入执行器` 增加非拥有有序参与者组重载，推荐机械形态为 `std::span<结构写入事务参与者* const>`。入口拒绝空指针、重复指针和无效回调；既有单参与者重载构造单元素视图并委托同一算法，不复制第二套收口逻辑。

固定顺序：

```text
业务回调形成结构候选并请求提交
-> 参与者按登记顺序逐项准备
-> 结构会话确认
-> 参与者按登记顺序逐项确认待发布
-> 全部确认后结构会话与参与者完成最后发布
```

回调异常、会话失败、未请求提交、准备失败、会话确认失败、参与者确认失败、故障注入或最后发布前故障时，参与者按登记逆序撤销，再撤销结构会话；任一撤销失败标记事务域隔离。单个参与者成功不等于事务成功。故障自检必须能定位每个参与者准备 / 确认序号，并证明逆序撤销顺序。

### 5.2 4170 专用发布记录账

`特征批次发布记录` 至少保存：强类型批次身份、非零侧账记录格式版本、受支持规则版本、宿主完整句柄、从 0 连续的全部有序项目、项目种类、特征定义 / 槽位 / 新值 / 来源完整句柄、换代写前当前关系 / 当前值完整句柄及原始值版本、发布后新当前关系完整句柄。记录格式版本只解释侧账字段形状，业务规则版本只解释批次语义，二者不得合并。不得保存原始 I64 / Vec 值副本，不得拆分句柄，不得在任何主信息槽写批次字段。

`特征批次发布记录账` 由 `特征体系数据操作` 生命周期拥有。访问器只返回：未找到、可读、许可竞争、未知格式、内部不一致。事务外预读未找到后，执行器必须先取得覆盖全部参与结构的唯一写入权，并在第一结构写前再次读取侧账：同义不形成候选并释放写入权后历史读回，异义零写冲突，异常保持原分类，只有仍未找到才形成候选。参与者在准备阶段取得侧账独占占用并最终检查同键竞争，登记一个未公开候选；确认待发布后才可进入最后发布；撤销删除本次候选；完成发布后记录不可变并可读。未知格式、顺序缺口、悬空引用或不完整记录不得读成历史成功。

### 5.3 来源、规则版本和摘要

来源版本只使用来源完整句柄中的版本。已发布来源必须与当前节点记录版本一致；同会话来源必须由结构会话证明是本事务候选且类型、归属和登记身份一致。不得存在第二个裸来源版本字段。

v0.2 只接受计划冻结的规则版本 1；零值和未知版本写前拒绝。调用方请求、公开服务结果和 4170 权威记录均不得携带可裁决载荷摘要。实现如保留运行期摘要，只能从强类型完整有序字段确定计算并始终回到完整比较；摘要相同、不同、清空、碰撞或重建均不得独立改变结果。

### 5.4 幂等和错误分类

同一批次身份再次提交：

| 账读取 / 比较 | 结果 |
| --- | --- |
| 未找到 | 继续全部写前准入，并在参与者准备阶段再次检查竞争 |
| 完整有序语义相同且结构互证 | 返回同一已发布记录、槽位、值、关系和来源 |
| 任一规则、宿主、数量、顺序、种类、定义、槽位、新值、来源或写前证据不同 | 幂等冲突，零写入 |
| 许可竞争 | 许可拒绝，不得返回已存在或冲突 |
| 来源 / 当前值版本漂移 | 版本漂移，零写入 |
| 未知规则或账格式 | 写前拒绝或未知格式，不得返回历史成功 |
| 已存在记录不完整、悬空或与结构不互证 | 内部逻辑错误，停止依赖路径 |

### 5.5 同层授权桥

桥模块不 import 任一项目领域 / 核心模块，只用依赖类型模板完成“形成 private 参与包、把两个内部参与者交给需求任务方法数据操作已有执行器、读取候选结果”机械访问。桥不拥有批次、任务、会话、令牌、参与者、执行器、仓库或业务状态。

特征模块 import 桥并把唯一真实桥实体设为 private 友元；需求任务方法模块单向 import 特征模块与桥，并通过自己的 private 辅助入口形成真实消费。特征模块禁止 import、前置声明或跨模块 friend `需求任务方法数据操作`。授权桥 import 在生产代码中只能出现于这两个数据操作模块；自检必须经过需求任务方法数据操作的正式或 Debug 专用窄入口验证，不得直接 import 桥。

function 运行有界完整自检([string]$程序路径, [string]$配置标签) {
    $既有进程 = @(Get-Process -Name '海中鱼巣' -ErrorAction SilentlyContinue)
    if ($既有进程.Count -ne 0) { throw "$配置标签 自检开始前存在海中鱼巣.exe 进程" }
    $标识 = [Guid]::NewGuid().ToString('N')
    $标准输出 = Join-Path ([IO.Path]::GetTempPath()) ("DD05F0-$配置标签-$标识.stdout.log")
    $错误输出 = Join-Path ([IO.Path]::GetTempPath()) ("DD05F0-$配置标签-$标识.stderr.log")
    $进程 = Start-Process -FilePath $程序路径 -ArgumentList '--self-test-exit' -PassThru -WindowStyle Hidden -RedirectStandardOutput $标准输出 -RedirectStandardError $错误输出
    if (-not $进程.WaitForExit(180000)) {
        Stop-Process -Id $进程.Id -Force
        $进程.WaitForExit()
        if (@(Get-Process -Name '海中鱼巣' -ErrorAction SilentlyContinue).Count -ne 0) { throw "$配置标签 超时清理后仍有海中鱼巣.exe 残留" }
        throw "$配置标签 完整自检超过固定 180 秒；已只清理 PID $($进程.Id)"
    }
    if (@(Get-Process -Name '海中鱼巣' -ErrorAction SilentlyContinue).Count -ne 0) { throw "$配置标签 自检结束后仍有海中鱼巣.exe 残留" }

    return @($标准输出, $错误输出)
}
