## 1. 目标

以现行外设、观察、存在、场景、体素、任务和自我循环规范为准，重建从真实 D455 采样材料到生产事实与自我回流的完整设计链：

```text

-> L0—L3 材料成熟处理
-> 三种运行模式与四类强类型产品
-> 一个物理报告队列和四类索引
-> 任务域唯一消费、等待工作项与已承接引用
-> 不可变外设方法工作包
-> 观察 / 识别 / 扫描 / 跟踪本能方法
-> 像素簇、存在候选、状态 / 动态和场景事实提交
-> 已确认存在的体素融合、分解和父子模型
-> 场景体素快照、视觉先验与后继观察候选
-> 任务验证、需求结算、自我下一轮
```

## 3. 已验证事实

1. 当前代码只有 D455 四路同步帧、缓存和显式真实样本模式；没有生产连续消费。
2. `--d455-real-sample-exit` 与生产运行期互斥，不能作为生产接通证据。
3. 当前运行线程自检明确验证 D455、体素和真实外设未接入。
4. 当前没有 L1—L3、四类产品、报告队列、任务域唯一消费、外设工作包、观察四方法、像素簇 / 存在候选、体素融合或生产装配实现。
5. DD-01—DD-04 保留可复用业务合同，但建立在旧基础接口与旧施工计划上；DD-03 / DD-04 必须重绑节点直接任务、方法和筹办接口。
6. DD-05 当前为归档重定向；DD-06、DD-07、观察四方法和体素主题缺少完整有效设计链。
7. #352 明确不负责 D455、外设或体素生产接线。
