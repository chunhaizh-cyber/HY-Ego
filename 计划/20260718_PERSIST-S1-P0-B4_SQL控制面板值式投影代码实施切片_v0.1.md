## 1. 目标与依据

依据恢复流程图 v0.3、详细设计第 17 节和 #295 事实矩阵。本切片只让 SQL 审计与只读控制面板消费同身份租约导出的值式投影；数据库、界面、HTML、日志和显示继续只做人读 / 审计材料，不持可写仓库引用、不裁决机器事实、不参与恢复。

来源性质为独立只读重建 + MIG-18 新增约束，不计旧控制面板、旧数据库或旧持久化能力迁移。

## 2. 依赖与假定接口

#299 已提供运行期纯只读查询适配，#297 已提供有效租约。当前 `控制面板服务.h`、窗口和 SQL 适配仍从旧对象 / 结构统计来源取得材料。目标新增一个值式投影模块，把租约查询结果转换为既有控制面板 / SQL 只读 DTO；不得把 DTO 反向写回领域结构。
