## 8. 非成功返回与失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 参数版本 / 键 / 值域无效 | 逻辑内返回 | 写前拒绝，数量不变 |
| 键已绑定错误类型或异义结构 | 逻辑内返回 | 不覆盖、不删除、不重绑 |
| 许可竞争 / 并发已提交 | 逻辑内返回或幂等读回 | 保留原领域状态，显式重试或读回 |
| 独立服务步骤成功、后继未完成 | 未发布系统角色 | 不形成清单；已提交对象必须各自完整、稳定键可读 |
| 最后关系会话失败 | 逻辑内返回或追根因 | 本会话新关系精确撤销，不形成清单 |
| 前置通过后权威读回矛盾 | 追根因解决 | 停止依赖路径，不扫描修复、不切旧路径 |
| 历史模块转换造成结果漂移 | 追根因解决 | 不以编译通过代替行为回归，停止归档 |

无法证明无非法半结构、无角色清单误发布或宿主保持空时，停止实施并登记断点。
