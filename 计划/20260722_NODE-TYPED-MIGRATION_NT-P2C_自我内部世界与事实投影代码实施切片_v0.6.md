## 4. DTO、ABI 与模块所有权

规则版本固定为：

```cpp
自我结构规则版本 = 2;
自我结构关系规则版本 = 1;
自我恢复投影验证规则版本 = 1;
```

`海中鱼巣.领域.数据操作.节点直接存在场景` 是 `自我结构.数据.h` 的唯一导出所有者：模块先导入核心节点 / 关系 / 索引、结构执行器和共同结果，再在模块作用域导出该头。其它 P2C 模块只 import 唯一所有者。

共享 DTO 字段顺序固定为：

| 类型 | 精确字段 |
| --- | --- |
| `完整自我初始化请求` | `uint32_t 自我结构规则版本`；五个具名 `节点稳定主键`；`vector<节点句柄> 出生期内部世界成员组` |
| `完整自我结构值式材料` | `uint32_t 自我结构规则版本`；五项具名 `节点直接身份记录`；五条具名 `正式关系记录` |
| `内部世界成员值式材料` | `节点直接身份记录 成员`；`正式关系记录 角色2证据` |
| `内部世界成员组值式材料` | `节点直接身份记录 内部世界`；`vector<内部世界成员值式材料> 项目组` |
| `自我聚合读取值式材料` | 完整自我、成员、槽位、状态、动态五组值式材料 |
| `自我结构退役规格` | `uint32_t 自我结构规则版本`；`节点句柄 自我`；`uint32_t 预期节点版本`；`自我结构退役授权语境 授权语境` |
| `自我结构退役材料` | 退役前自我 / 内部世界；待失效关系 21 组；其它外部当前关系组；待移除索引组 |
| `自我结构退役审计材料` | 退役前自我 / 内部世界；发布后自我审计；发布后关系 21 / 外部关系审计组；已移除索引组 |
| `自我恢复投影验证请求` | `uint32_t 恢复投影规则版本`；自我结构候选组；成员、槽位、状态、动态组 |
| `自我恢复投影验证材料` | `自我聚合读取值式材料 投影` |

`自我结构退役参与包` 完整类和含 `optional<自我结构退役参与包>` 的形成结果在数据操作模块内定义，不放入共享头。

三个专用结果字段固定为：

```text
自我结构退役参与包形成结果
  状态: 自我结构退役参与包形成状态
  材料: optional<自我结构退役材料>
  参与包: optional<自我结构退役参与包>
  只有“已形成”同时携带材料和参与包

自我结构退役执行结果
  权威结构结果: 节点直接身份结构写入结果
  发布后读回状态: 合同结果状态
  发布后读回原因: 合同结果原因
  审计: optional<自我结构退役审计材料>

自我恢复投影验证结果
  状态: 自我恢复投影验证状态
  材料: optional<自我恢复投影验证材料>
  只有“已通过”携带材料
```

枚举数值固定为：

```cpp
enum class 自我结构退役授权语境 : std::uint32_t {
    非当前历史自我 = 1
};

enum class 自我结构退役参与包形成状态 : std::uint32_t {
    已形成 = 1,
    合法未找到 = 2,
    入口拒绝 = 3,
    许可竞争 = 4,
    版本漂移 = 5,
    资源失败 = 6,
    内部不一致 = 0x8000
};

enum class 自我恢复投影验证状态 : std::uint32_t {
    已通过 = 1,
    入口拒绝 = 2,
    材料不一致 = 3,
    资源失败 = 4,
    内部不一致 = 0x8000
};
```

结果别名固定为：

```cpp
using 完整自我结构写入结果 = 合同单项结果<完整自我结构值式材料>;
using 自我结构读取结果 = 合同单项结果<完整自我结构值式材料>;
using 内部世界成员组读取结果 = 合同具名组结果<内部世界成员组值式材料>;
using 自我聚合读取结果 = 合同单项结果<自我聚合读取值式材料>;
using 自我结构退役规格结果 = 合同单项结果<自我结构退役规格>;
using 自我结构退役材料读取结果 = 带值读取结果<自我结构退役材料>;
```

P2C 没有私有记录仓结构版本；核心多写集最终提交可能返回全零诊断槽。禁止使用要求非零结构版本的 `带值结构写入结果<T>`，也禁止在自我 / 成员 / 聚合材料中新增虚构顶层结构版本。
