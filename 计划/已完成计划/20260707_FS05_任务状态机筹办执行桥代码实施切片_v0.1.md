## 4. 目标服务和目标结构

| 项 | 目标口径 |
| --- | --- |
| 任务服务 | 提供生命周期迁移、筹办回执请求壳、执行桥请求材料和非权威统计读取入口 |
| 需求服务 | 只提供来源需求、目标状态、主体、目标宿主和场景材料 |
| 方法服务 | 只提供候选方法、方法首、条件 / 结果和动作入口角色材料 |
| 动态服务 | 后续记录方法调用动作动态，本切片不触发真实动作 |
| 非权威缓存 / 统计 | 承载运行统计、权重和错误码材料，不裁决任务完成 |

## 7. 实施步骤

```text
S0：复核任务服务当前生命周期、承接材料、实际结果状态和任务方法关系入口。
S1：实现生命周期状态迁移请求入口和拒绝矩阵。
S2：实现任务筹办回执只读材料和提交请求壳，构造回执不写任务状态、方法关系或需求树。
S3：实现执行桥请求材料和依赖复核入口，缺任务方法关系、方法结构或动作入口时拒绝。
S4：实现任务运行统计、权重、错误码非权威材料分流入口。
S5：补默认入口验收并同步断点清单、项目记忆和验证结果。
```
