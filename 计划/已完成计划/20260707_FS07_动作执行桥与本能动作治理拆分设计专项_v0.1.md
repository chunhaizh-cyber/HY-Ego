
## 4. 目标服务和目标结构

| 项 | 目标口径 |
| --- | --- |
| 方法服务 | 继续承载动作入口角色和稳定动作键候选读取 |
| 任务服务 | 后续只提供任务授权、任务选择方法和任务状态材料 |
| 动态服务 | 记录动作动态证据材料，不执行动作 |
| 因果服务 | 只接受来源动态准入，不持久化来源动态关系 |
| 关系仓库 | 承载动作入口归属、状态、输入 / 输出规格和来源关系 |
