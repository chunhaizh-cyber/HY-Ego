```text
入口自检专用删除、生产初始化补偿、领域正常删除、结构事务包、历史兼容清理。
调用时是否已有同域许可、是否可能无令牌重入、是否跨线程、是否持仓库 / 活动图锁。
```

每个调用点只能归入以下一类：

| 分类 | 含义 | 后继裁决 |
| --- | --- | --- |
| 测试隔离材料 | 只在独立自检夹具发生 | 保持自检所有权，不进入生产接线 |
| 写前拒绝前无结构 | 尚未写入，无清理 | 不生成补偿计划 |
| 创建期可定位补偿 | 本次新建结构可精确逆序清理 | 判断是否需要同域独占或预构造包 |
| 已发布权威结构删除 | 删除既有机器事实 | 必须由结构所有者 / 领域服务裁决 |
| 候选 / 非权威材料丢弃 | 不改变权威结构 | 保持逻辑内返回 |
| 内部不一致后清理 | 前置已满足后写入失败 | 先归追根因；不得把补偿冒充正常返回 |
