## 1. 定位

本切片只实现一般用途、因果预测用途、因果行动用途三类非权威观察缓存，以及概念图服务结合活动身份、实例支持和外部引用读取形成的生命周期复核候选。

本切片不直接改变生命周期，不创建命名需求，不自动评定稳定因果，不物理删除概念。

## 5. 假定接口契约

目标接口：

```text
缓存类型::概念用途观察
来源结构类型::概念
缓存值类型::用途观察
统计服务::记录概念用途观察(...)
统计服务::读取概念用途观察(...)
统计服务::清空概念用途观察(...)
概念图服务::记录活动概念用途观察(..., 统计服务)
概念图服务::读取生命周期复核候选(..., 统计服务)
```

契约：

```text
一般用途可用于四类活动概念，效用累计保持 0。
预测与行动用途只接受因果概念，记录和命名空间互相隔离。
同一命名空间、概念和用途类型的时间戳不得倒退。
缓存始终不可持久化、不可裁决身份 / 生命周期 / 因果事实，可清空和按规则版本失效。
概念图服务重读活动版本、当前生命周期、实例支持和外部引用后才形成候选。
候选只包含保持活跃、建议冷却、建议退役复核、建议重新激活或材料不足，不写关系。
```

## 7. 实施步骤

### S6B-0 实际接口复核

比对 #194 实际生命周期接口、统计服务现有缓存锁和 #189 可能形成的缓存扩展，不重复实现同功能锁或命名空间工具。

### S6B-1 统计 DTO 与缓存

新增三类用途及观察材料；以完整概念句柄、用途类型和完整命名空间为键，写入时先复核时间顺序，再原子更新次数、最近时间和预测 / 行动效用累计。

### S6B-2 概念服务封口

概念图服务拒绝非活动概念和类别不匹配；读取候选时组合用途材料、生命周期、活动支持和外部引用，不让统计服务读取或写权威概念关系。

### S6B-3 清空与规则变化

清空指定命名空间或全部概念用途缓存后，概念图节点、三类图关系、生命周期关系和活动版本保持不变。

### S6B-4 自检

覆盖三类隔离、时间倒退、并发不同概念 / 用途、缓存清空、规则版本变化、有支持 / 外部引用时不建议冷却和候选不写生命周期。

## 10. 入口拒绝

```text
命名空间无效、可持久化或类型组合不匹配。
概念句柄无效、非活动或活动版本不一致。
一般用途携带非零效用变化。
非因果概念提交预测 / 行动用途。
时间戳为 0、超过 I64 上限或小于同键最近时间。
累计次数 / 效用溢出。
复核规则版本、当前时间或截止时间无效。
```

拒绝后缓存和全部权威结构不变化。

## 11. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 无效命名空间、概念、用途或时间 | 逻辑内返回 | 不写缓存 |
| 非因果预测 / 行动观察 | 逻辑内返回 | 不生成候选 |
| 缓存缺失或规则版本不同 | 逻辑内返回 | 返回材料不足，不改变生命周期 |
| 有支持、外部引用或近期有效用途 | 正常候选 | 保持活跃 / 建议激活 |
| 前置通过后缓存写入或读回不一致 | 追根因解决 | 停止 S6B 依赖路径 |
| 候选读取意外改变节点 / 关系 / 活动版本 | 追根因解决 | 定位越权写入根因 |
