### S2：分片锁与创建复用

```text
固定 64 个分片互斥量，以完整概念句柄稳定选择分片。
外层先取得 #217 同域独占许可，再取得目标概念分片锁；全部内部仓库调用显式复用同域令牌，不递归取得共享许可。
许可前只复核消息协议、专用类型和值式字段完整性，不调用概念图、语素或接域仓库。
分片锁内先调用概念图令牌重载；概念图持活动图读锁，经令牌生命周期辅助和语素令牌重载复核当前请求、活动版本及名称入口为空。
当前请求消失、活动版本漂移或名称已经出现时，释放分片锁与独占许可后逻辑内返回；不得进入需求服务写路径。
独占许可和分片锁内重新读取并分类未完成需求数量。
零个进入创建；一个完整复用；多个触发追根因且不返回第一个。
新活动版本请求仍复用同一未完成需求，活动版本不持久化为需求身份。
同概念并发只允许一个已创建，不同概念保持可并发。
```

## 7. 入口拒绝

```text
主体或目标宿主不是存在，场景不是场景。
共享目标状态无效、不是抽象状态或与初始化结果不一致。
目标概念无效、没有当前概念身份或目标关系类型不受支持。
服务根不是完整服务根需求。
主体 / 宿主 / 场景与服务根初始化上下文不一致。
同概念已经存在多个未完成需求。
既有唯一候选缺承接、目标关系、目标状态、父需求或结算材料。
消息协议拒绝；或在独占许可与分片锁内令牌重读后确认活动版本漂移、当前请求消失或名称已经存在。
```

所有输入拒绝发生在创建节点和关系之前；多需求或内部残缺归追根因，不得普通拒绝后继续。

## 8. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 输入类型、初始化上下文或消息值形状不满足 | 逻辑内返回 | 许可前不读活动当前态，不写结构 |
| 许可内令牌重读确认请求消失、活动版本漂移或名称已出现 | 逻辑内返回 | 不进入需求服务，释放分片锁与独占许可 |
| 一个完整未完成需求 | 幂等复用 | 不推进关系版本，不新增父子 |
| 多个未完成需求或既有当前结构残缺 | 追根因解决 | 不选第一项，不创建新需求 |
| 新候选节点、承接关系或目标关系写入不及预期 | 追根因解决 | 逆序撤销本次未发布候选 |
| 服务根挂载不及预期或发布后完整读回矛盾 | 追根因解决 | 删除本次父子并逆序撤销，停止依赖 |
| 删除补偿本身失败或删除后仍可读 | 追根因解决 | 立即停止，不以扫描器继续修复 |
