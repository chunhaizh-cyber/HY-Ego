## 2. 确认范围

本计划确认后，只允许第一轮保守实现：

```text
控制面板只读功能服务壳。
控制面板只读总览材料。
六类树视图候选材料：世界树、概念树、语素树、需求树、任务树、方法树。
状态 / 动态作为详情材料的边界，不作为控制面板树分类。
操作按钮或命令请求壳，只生成请求材料，不执行写入。
数据库 / SQL / ADO 审计适配候选门禁。
WebView / HTML / JSON / 截图渲染通道候选门禁。
D455 / 体素 / 外设独立专项门禁。
拒绝后权威结构不变化验收。
```

## 3. 明确排除项

```text
不实现真实控制面板 UI。
不接 WebView、HTML 渲染窗口、JSON 接口或截图入口。
不接 SQL、ADO 或数据库运行期裁决。
不实现数据库审计适配服务。
不接 D455、体素或外设显示。
不让按钮、命令、显示标题、HTML、JSON、截图、日志或统计裁决机器事实。
不写节点、主信息、关系、索引、特征、特征值、需求、任务、方法、状态、动态或因果结构。
不新增只为显示方便存在的机器字段。
不修改核心仓库和既有领域服务写入口。
不迁移旧控制面板函数体。
不声明控制面板、数据库、WebView、外设显示或旧显示能力迁移完成。
```

## 4. 目标服务 / 目标结构

```text
目标服务：控制面板服务。
服务定位：只读人读材料组织方和操作请求壳生成方，不是机器事实写入方。
只读来源：显示服务或后续已确认领域服务只读入口。
输出材料：控制面板只读材料、控制面板树视图材料、控制面板操作请求材料、控制面板门禁结果材料。
结构变化：本轮不得改变权威机器结构；只允许新增服务壳和入口自检样例。
```

若实施时发现需要世界树、概念树、语素树等更细 DTO 契约，第一轮只能返回空候选 / 门禁材料或只读占位材料，不得绕过对应领域服务补写结构。

## 7. 实施步骤

```text
S0：复核显示服务第一轮只读边界、控制面板详细设计和排除项证据。
S1：新增控制面板服务壳，入口参数先拒绝越权读取、写事实、文本裁决、SQL 裁决和外部通道接入请求。
S2：新增只读总览材料入口，只能读取显示服务材料或保守空材料，不直接读写仓库内部对象。
S3：新增六类树视图候选材料入口；状态、动态只能作为详情材料，不作为树分类。
S4：新增操作请求壳入口；按钮或命令只生成请求材料，不调用写业务事实入口。
S5：新增数据库审计、渲染通道、D455 / 体素 / 外设后续专项门禁返回材料。
S6：补默认入口验收，覆盖 CONSOLE-PANEL-A1 到 CONSOLE-PANEL-A9。
S7：同步计划索引、项目记忆、验收记录和实施断点清单。
```

## 8. 验收方式

计划生成阶段只验证文档治理：

```powershell
git diff --check
python .\tools\check_specs.py
```

本计划确认并执行后，实施窗口必须至少验证：

```powershell
git diff --check -- 海中鱼巣/领域/控制面板服务.h 海中鱼巣/入口.cpp
python .\tools\check_specs.py
msbuild .\海中鱼巣.vcxproj /p:Configuration=Debug /p:Platform=x64 /m
.\x64\Debug\海中鱼巣.exe
rg -n "SQL|ADO|WebView|D455|体素|外设|std::thread|std::async|工作队列" 海中鱼巣/领域/控制面板服务.h 海中鱼巣/入口.cpp
rg -n "写节点|写主信息|写关系|写索引|裁决需求|裁决任务|裁决方法" 海中鱼巣/领域/控制面板服务.h
```

默认入口验收项：

| 编号 | 验收项 | 通过口径 |
| --- | --- | --- |
| CONSOLE-PANEL-A1 | 服务只读来源 | 控制面板只读总览只能来自显示服务或保守只读材料 |
| CONSOLE-PANEL-A2 | 越权读取拒绝 | 绕过服务读取仓库内部对象的请求被拒绝 |
| CONSOLE-PANEL-A3 | 写事实拒绝 | 按钮或命令直接写业务事实请求被拒绝 |
| CONSOLE-PANEL-A4 | 文本裁决拒绝 | HTML / JSON / 标题 / 日志不得裁决机器事实 |
| CONSOLE-PANEL-A5 | 树视图分类 | 树分类仅包含世界树、概念树、语素树、需求树、任务树、方法树 |
| CONSOLE-PANEL-A6 | 状态动态边界 | 状态、动态只能作为详情材料，不作为树分类 |
| CONSOLE-PANEL-A7 | SQL 边界 | SQL / ADO 裁决运行期事实请求被拒绝 |
| CONSOLE-PANEL-A8 | 外部通道门禁 | WebView / 截图 / D455 / 体素 / 外设请求转后续专项 |
| CONSOLE-PANEL-A9 | 结构不变 | 控制面板请求前后权威结构计数或句柄组不变 |
