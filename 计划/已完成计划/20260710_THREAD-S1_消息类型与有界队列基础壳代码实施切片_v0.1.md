## 4. 目标结构

| 结构 | 第一轮用途 | 权威性 |
| --- | --- | --- |
| 运行消息类型 | 区分调度周期、停止、任务执行请求、任务拒绝回执、结果回执、缓存刷新、事件日志、外部材料等消息 | 非事实 |
| 线程角色 | 标注运行宿主、任务管理、任务工作、缓存统计、事件日志、外设采样材料等来源或目标角色 | 非事实 |
| 消息优先级 | 支持停止消息最高优先级和普通消息排序材料 | 非事实 |
| 运行消息 | 承载请求、材料、回执和调度信号 | 非事实 |
| 有界消息队列 | 固定容量、局部失败、停止唤醒和基础并发友好入口壳 | 工程承载，不裁决机器事实 |
| 幂等 / 顺序材料 | 为后续复核提供幂等键、批次号、任务编号和任务序号材料 | 材料，不裁决业务结果 |

## 10. 验收矩阵

| 编号 | 验收项 | 通过标准 |
| --- | --- | --- |
| THREAD-S1-A1 | 消息不承载事实 | 消息结构只保存请求、材料、回执、调度信号和复核材料，不保存任务完成、需求满足、方法成功或世界事实 |
| THREAD-S1-A2 | 队列有界 | 固定容量队列满载时返回局部失败，不写业务失败事实 |

| THREAD-S1-A4 | 停止后拒绝 | 停止后普通消息入队被拒绝，拒绝结果不改变业务结构 |
| THREAD-S1-A5 | 幂等冲突拒绝 | 同一幂等键不同材料摘要被识别为冲突材料，不合并为机器事实 |
