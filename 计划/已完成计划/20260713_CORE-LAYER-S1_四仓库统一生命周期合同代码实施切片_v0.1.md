## 1. 当前事实与假定契约

正式 #262 矩阵已证明可在不删除现有 ABI 的情况下增加统一结构结果和自检，同时确认三项真实漂移：主信息 / 节点候选记录创建即有效；关系 / 索引没有候选类型；关系 / 索引旧删除只验证共享路径。JY-317 已给出默认裁决，执行时不得另行扩大。

## 2. 核心处理

统一四仓库对入口拒绝、许可拒绝、版本漂移、内部不一致、确认、撤销和普通删除的结构化结果；固定主信息->节点->关系/索引创建依赖、端点有效性、编号不复用、版本推进和候选幂等。主信息 / 节点候选不新增仓库记录状态；关系 / 索引不新增候选类型。为四仓库形成独占严格普通删除入口，现有关系 / 索引共享删除和其它 bool / optional 入口原义保留为兼容路径。

不新增业务节点类型，不把需求 / 任务 / 方法准入下沉仓库，不实现关系二级索引或性能计数。
