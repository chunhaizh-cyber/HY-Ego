## 4. 目标服务和目标结构

| 项 | 目标口径 |
| --- | --- |
| 世界服务 | 兼容壳用途可枚举，已明确类型的写入优先走原子服务 |
| 语素服务 | 只绑定已有长期基础信息节点，不绑定临时实例状态 / 动态 |
| 状态服务 | 提供抽象状态候选、实例状态读取拒绝和引用阻断材料 |
| 动态服务 | 提供抽象动态候选、实例动态读取拒绝和引用阻断材料 |
| 二次特征 / 因果服务 | 只提供候选材料读取，不被本切片改写生命周期规则 |
