S0 记录 `$计划段起点 = (git rev-parse HEAD).Trim()`；S4 的两个 diff check 均须退出 0。S5 后运行：

```powershell
$期望 = @('海中鱼巣/适配/处理.D455稳定观察.ixx','海中鱼巣/适配/自检.D455稳定观察.ixx','实施记录/20260724_D455-P4_PER-C4稳定门代码实施_Codex断点清单.md') | Sort-Object

if (Compare-Object $期望 $实际) { exit 1 }
if ((@(rg -n "D455稳定观察ABI版本\s*=\s*2" '海中鱼巣/适配/处理.D455稳定观察.ixx')).Count -ne 1) { exit 1 }
if ((@(rg -n "推进D455稳定窗口" '海中鱼巣/适配/处理.D455稳定观察.ixx')).Count -lt 1) { exit 1 }
```

`git diff --check`、`git diff --cached --check` 和上段命令均须退出 0；ABI 精确 1 处。
