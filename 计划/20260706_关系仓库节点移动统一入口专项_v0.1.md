# 关系仓库节点移动统一入口专项

编号：HR-001-MOVE
日期：2026-07-06
状态：v0.1 / 已实现 / 已验证

## 1. 目标

把“将一个节点移动到另一个节点下”统一解释为普通父子关系重挂，避免每个领域服务重复实现移动逻辑。

## 2. 口径

```text
通用移动入口：关系仓库::重挂节点(节点, 新父节点)
适用范围：普通父子关系
不适用范围：归属、引用、模板、运行期临时、语素对应、语素概念追溯等非普通父子关系
```

领域服务不得复制移动算法。需要移动语义时，只能做本领域前置拒绝后调用通用入口。

## 3. 本切片修改

```text
增强 关系仓库::重挂节点 入口拒绝。
补入口样例验证移动到自身、移动到子孙、移动到当前父节点会拒绝。
同步关系仓库详细设计、项目规则、函数说明和项目记忆。
```

## 4. 验收

