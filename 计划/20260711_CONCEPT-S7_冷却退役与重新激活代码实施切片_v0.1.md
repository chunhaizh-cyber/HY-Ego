本切片只实现活动概念的活跃到冷却、冷却到退役、冷却 / 退役到活跃的生命周期转换，以及只返回活跃概念的主动召回读取入口。

退役概念仍保留节点、签名、名称、上下位和树投影可读性。本切片不物理删除、不创建命名需求、不由统计直接改状态。

## 5. 假定接口契约

目标接口：

```text
关系仓库::获取节点相关关系记录组(节点)
关系仓库::重挂关系并返回新句柄(关系, 新源, 新目标)
概念图服务::转换概念生命周期(概念, 目标阶段, 候选, 规则)
概念图服务::读取主动召回概念节点组()
```

契约：

```text
关系仓库只返回有效端点的值式记录副本，不解释外部引用或生命周期。
生命周期转换复用同一关系记录，原子更新目标和版本，不先删后建。
根概念始终活跃；活跃不能直接退役；退役不能转冷却；同状态幂等不推进版本。
冷却 / 退役前重读实例支持、外部正式引用、用途候选、活动版本和当前状态。
重新激活可由新实例支持、外部正式引用或有效用途触发，保留原概念句柄和签名。
活动概念读取仍返回全图；主动召回读取只返回活跃概念，退役概念仍在树投影中可见。
```

## 7. 实施步骤

### S7-0 实际接口复核

逐项比对 #194、#195 实际接口和 #192 对关系仓库的扩展；复用已有通用结果和锁，不复制算法。

### S7-1 关系仓库值式能力

新增节点全部有效相关关系记录读取；如现有 `重挂关系` 无法安全获得新句柄，则新增兼容封口并让旧入口保持语义。

### S7-2 权威复核

概念图服务区分图内部关系和外部正式引用；实例支持单独复核。任何有效外部引用或实例支持阻止冷却 / 退役。

### S7-3 状态转换

在生命周期写锁内复核状态机、候选和权威材料，再原子重挂唯一关系。读回目标、版本和活动身份后返回成功材料。

### S7-4 重新激活与召回

冷却 / 退役概念在新支持、引用或有效用途出现时重挂为活跃。主动召回按生命周期过滤，既有树投影继续保留退役节点。

### S7-5 并发和回归

覆盖同一概念相反转换、支持关系与退役请求并发、根保护、非法跳转、同状态幂等和 S1-S6B 回归。

```text

根概念请求离开活跃。
```

## 11. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 非法状态跳转、根保护、候选不足 | 逻辑内返回 | 保持原状态 |
| 同状态请求 | 幂等逻辑内返回 | 不推进关系版本 |
| 仍有支持 / 外部引用 / 有效用途 | 逻辑内返回 | 保持活跃或重新激活 |
| 前置通过后关系重挂失败 | 追根因解决 | 停止 S7 依赖路径 |
| 重挂后新句柄、目标、版本或唯一性读回不一致 | 追根因解决 | 不发布成功转换材料 |
| 主动召回返回冷却 / 退役概念 | 追根因解决 | 定位生命周期过滤根因 |
| 树投影因退役丢失概念 | 追根因解决 | 不把退役冒充删除 |
