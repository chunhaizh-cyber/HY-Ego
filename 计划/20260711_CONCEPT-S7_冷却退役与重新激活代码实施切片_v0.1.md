## 1. 定位

本切片只实现活动概念的：

```text
活跃 -> 冷却
冷却 -> 退役
冷却 / 退役 -> 活跃
主动召回只返回活跃概念
```

退役概念仍保留节点、签名、名称、上下位和树投影可读性。本切片不物理删除、不创建命名需求、不由统计直接改状态、不负责判断实例支持何时失效。

## 5. 假定接口契约

#210 预期提供：

```text
关系仓库::获取节点相关关系记录组(节点)
关系仓库::重挂关系并返回新句柄(关系, 新源, 新目标)
概念图服务::失效活动实例支持(请求)
概念图服务::读取实例支持审计组(实例, 概念)
活动支持失效后普通支持投影移除且活动版本推进
```

#197 目标接口：

```text
概念图服务::转换概念生命周期(概念, 目标阶段, 候选, 规则)
概念图服务::读取主动召回概念节点组()
```

契约：

```text
生命周期转换复用同一概念生命周期关系，原子更新目标和版本，不先删后建。
根概念始终活跃；活跃不能直接退役；退役不能转冷却；同状态幂等不推进版本。
冷却 / 退役前重读当前有效实例支持、外部正式引用、用途候选、活动版本和当前状态。
已失效支持只在审计入口可读，不计入活性。
重新激活可由新有效支持、外部正式引用或有效用途触发，保留原概念句柄和签名。
活动概念读取仍返回全图；主动召回只返回活跃概念，退役概念仍在树投影中可见。
```

## 7. 实施步骤

### S7-0：草案恢复与实际接口复核

```text
确认 #210 已完成、提交、推送并形成正式实施记录。
恢复 #197 草案后逐项适配 #210 接口，不保留重复仓库实现。
删除草案中已由 #210 承接的关系新句柄 / 相关关系读取重复代码。
重新读取 #194、#195、#210 正式接口和全部调用点。
```

### S7-1：真实可达样本

```text
从当前活动图选择一个真实普通概念，禁止回退四根。
通过 #195 正式观察入口形成已过最近使用截止时间、但观察次数非零的历史用途材料。
通过 #210 正式入口逐条失效该概念全部当前有效支持。
确认活动版本推进、有效支持为零、审计历史可读、节点 / 签名 / 树投影仍存在。
重新生成 #195 候选，旧候选必须过期，新候选才可进入 S7。
```

### S7-2：权威复核

```text
概念图服务使用 #210 的节点相关有效关系读取区分图内部关系和外部正式引用。
实例支持只读取当前活动投影中的有效关系。
任一有效支持、外部正式引用、正向用途或近期用途继续阻止冷却 / 退役。
无有效支持但观察次数为零仍是材料不足，不得自动冷却。
```

### S7-3：状态转换

```text
在生命周期写串行边界内复核状态机、候选活动版本、候选规则和当前权威材料。
调用 #210 的值式重挂入口，把唯一生命周期关系原地重挂到目标阶段状态。
读回同一关系编号、新版本、目标状态、阶段和活动概念身份后返回成功。
```

### S7-4：重新激活与召回

```text
冷却 / 退役概念在新有效支持、外部正式引用或当前有效用途出现时可重新激活。
主动召回按当前生命周期过滤，只返回活跃概念。
活动全图和抽象树投影继续返回冷却 / 退役节点，显示层只读。
```

### S7-5：并发和回归

覆盖同一概念相反转换、支持失效与冷却请求、支持新建与退役请求、根保护、非法跳转、同状态幂等和 S1-S6D 回归。

## 10. 入口拒绝

```text
概念无效、非活动、图版本不一致或生命周期不唯一。
目标阶段未定义或状态节点未登记。
根概念请求离开活跃。
活跃直接退役、退役转冷却或其它非法跳转。
候选规则版本、截止时间或当前时间无效。
冷却 / 退役时仍有当前有效实例支持、外部正式引用、正向用途或近期用途。
候选活动版本与 #210 支持失效后的当前活动版本不一致。
观察次数为零而试图把材料不足改写成建议冷却。
```

拒绝后生命周期关系目标 / 版本、活动图版本和所有仓库数量不变化。

## 11. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 非法状态跳转、根保护、候选不足、旧活动版本 | 逻辑内返回 | 保持原状态 |
| 同状态请求 | 幂等逻辑内返回 | 不推进关系版本 |
| 仍有有效支持 / 外部引用 / 有效用途 | 逻辑内返回 | 保持活跃或允许重新激活 |
| 无支持但观察次数为零 | 逻辑内返回 | 返回材料不足，不冷却 |
| 前置通过后生命周期关系重挂失败 | 追根因解决 | 停止 S7 依赖路径 |
| 重挂后新句柄、目标、版本或唯一性读回不一致 | 追根因解决 | 不发布成功转换材料 |
| 主动召回返回冷却 / 退役概念 | 追根因解决 | 定位生命周期过滤根因 |
| 树投影因退役丢失概念 | 追根因解决 | 不把退役冒充删除 |
