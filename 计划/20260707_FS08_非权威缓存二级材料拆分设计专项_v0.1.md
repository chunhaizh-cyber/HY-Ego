## 4. 目标服务和目标结构

| 项 | 目标口径 |
| --- | --- |
| 统计服务 | 继续承载非权威缓存和结构统计快照入口 |
| 特征服务 | 后续特征值材料只能经特征服务提供，统计服务不得直接依赖特征值服务 |
| 需求服务 / 任务服务 | 只提供已确认权威结构材料，缓存摘要不得回写业务事实 |
| 索引仓库 | 可用于候选召回或缓存键，但不裁决事实 |
| 日志 / 显示 | 只做人读材料，不参与缓存裁决 |
