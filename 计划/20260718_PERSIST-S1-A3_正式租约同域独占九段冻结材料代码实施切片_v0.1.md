## 1. 目标与依据

依据恢复流程图 v0.3 当前 JY-441 替换子图、消费者流程图 v0.7、详细设计第 17 / 20—22 节、#294 / #295 矩阵和 #304—#306 预期正式产物。本切片从 #297 / #298 显式生产运行期实际持有的正式租约取得同域独占许可，按固定参与者顺序只复制权威材料，释放许可后形成不可变九段值式材料。排序、编码、校验和、文件 I/O、外部解析和恢复均不在本切片。

来源性质为 MIG-18 重新实现 / 新增约束 / 全新能力，不计旧能力迁移。

## 2. 依赖门控与假定接口

```text
#297 / #298：显式生产运行期具有唯一上下文 / 宿主 / 同身份租约和完整 B0 装配；#303 默认切换不是冻结实现前置。
#304：索引所有者声明完整，默认新运行期未知所有者为 0。
#305：四仓库全部状态与精确高水位导出。
#306：特征值、系统角色、领域登记和概念活动导出。
```

假定唯一公开入口为值式：

```text
形成权威冻结材料(运行期上下文租约, 快照生成身份) -> 权威冻结结果
```

入口不得暴露原始令牌 / 许可。运行期上下文内部使用自身接线取得独占许可；冻结模块不得新建协调器、仓库或平行上下文。

参与者固定顺序：节点 -> 主信息 -> 关系 -> 索引 -> 特征值 -> 系统角色 -> 领域登记 -> 概念活动。DTO 形成 HYSNAP01 九段的完整值式来源；O1 只通过四仓库和领域登记体现，不新增第十段。
