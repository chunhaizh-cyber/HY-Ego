## 5. 运行与证据保存

每一轮必须先把完整输出写入独立文件，再读取退出码和叶子；禁止只保留末尾若干行：

```powershell
$证据根 = 'D:\海中鱼巣\日志\诊断\TASK-RESULT-STABILITY-S0'
New-Item -ItemType Directory -Force -Path $证据根 | Out-Null

foreach ($轮 in 1..60) {
    $轮日志 = Join-Path $证据根 ("baseline-round-{0:D2}.log" -f $轮)
    & .\x64\Debug\海中鱼巣.exe --self-test-exit *> $轮日志
    $退出码 = $LASTEXITCODE
    $叶子 = @(Get-Content -LiteralPath $轮日志 |
        Select-String '^TASK-RESULT-S1-A(0[1-9]|1[0-8]) ')
    $失败叶子 = @($叶子 | Where-Object { $_.Line -match ': 失败$' })

    if ($叶子.Count -ne 18 -or $退出码 -ne 0 -or $失败叶子.Count -ne 0) {
        break
    }
    Start-Sleep -Seconds 2
}
```

诊断记录必须保存每轮退出码和叶子计数；发生失败时逐字保存全部失败叶子、该轮 18 行 TASK-RESULT 输出、顶层失败门禁和原始日志 SHA-256。不得只写“任务结果结算失败”。
