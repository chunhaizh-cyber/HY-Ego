## 2. 已验证事实与接口合同

#339 当前真实公开面已有按源读取、关系创建 / 失效 / 重挂、索引创建、节点读取、结构会话和多参与者执行器；没有本计划五个目标入口。

完整签名固定为：

```cpp
std::vector<正式关系记录> 读取指向有效关系组(
    节点句柄 目标节点, 关系类型 类型) const;
std::vector<正式关系记录> 读取相关有效关系组(
    节点句柄 节点, 关系类型 类型) const;

    关系类型 类型, std::int64_t 顺序号) const;

    节点句柄 节点);
节点直接身份带值结构写入结果<可重建索引记录> 移除索引候选(
    const 索引物理键& 键);
```

前三个入口同时由正式关系仓库和结构会话一致视图提供；候选叠加、排序、去重、有效性、逻辑内拒绝和内部错误按详细设计执行。
