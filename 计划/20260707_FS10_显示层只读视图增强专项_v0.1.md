## 1. 任务定位

本计划承接 FS-10 显示层只读候选只读扫描和逻辑迁移包归并材料，目标是把可进入第一轮增强的显示标题、世界结构只读视图材料和控制面板快照 DTO / JSON 边界转成后续可执行路线。

本计划是确认对象。`实施记录/20260707_FS10_显示层只读候选逻辑迁移包归并表.md` 只作为生成本计划的证据材料，不是本窗口最终确认目标。

本计划不迁移旧函数体，不新增旧主信息字段，不直接进入 C++ 实施。代码实施必须另建切片，并再次确认允许文件、禁止文件和验证方式。

## 5. 设计条款清单

### 必须实现

```text
显示标题只能由只读视图派生，不写回节点、主信息、关系或索引。
显示标题不得作为语素绑定、候选裁决、节点身份或业务状态依据。
世界结构只读视图只能通过领域服务或授权只读仓库入口读取材料。
只读视图不得暴露可写引用、可写指针或可写内部容器。
只读 DTO / JSON 只做人读材料，不参与需求满足、任务完成、方法成功、动作动态或因果结论。
读取失败或材料缺失只能返回空视图或错误视图，不得补写业务结构。
```

### 禁止实现

```text
不接 SQL 控制面板显示镜像。
不实现控制面板 HTML / WebView2 / 窗口 / 截图。
不实现控制面板摘要线程或任务管理界面线程。
不读取或写入 D455 / 体素 / 外设材料。
不把显示标题、JSON、HTML、控制台摘要或线程快照作为机器事实。
不迁移旧控制面板函数体。
```

### 后续待补

```text
显示层只读视图详细设计。
只读视图 DTO / JSON 格式计划。
只读视图入口拒绝验收矩阵。
控制面板 / WebView2 后续专项。
SQL 只读投影后续专项。
外设 / 体素显示材料后续专项。
```
