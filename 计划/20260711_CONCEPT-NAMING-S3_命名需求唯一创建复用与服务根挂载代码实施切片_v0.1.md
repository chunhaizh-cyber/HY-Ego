## 8. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 输入类型、初始化上下文、消息或活动版本不满足 | 逻辑内返回 | 不写结构 |
| 一个完整未完成需求 | 幂等复用 | 不推进关系版本，不新增父子 |
| 多个未完成需求或既有当前结构残缺 | 追根因解决 | 不选第一项，不创建新需求 |
| 新候选节点、承接关系或目标关系写入不及预期 | 追根因解决 | 逆序撤销本次未发布候选 |
| 服务根挂载不及预期或发布后完整读回矛盾 | 追根因解决 | 删除本次父子并逆序撤销，停止依赖 |
| 删除补偿本身失败或删除后仍可读 | 追根因解决 | 立即停止，不以扫描器继续修复 |

## 9. 执行前接口复核与漂移退回

```text

#217 没有形成可由需求服务复用的同域独占许可 / 令牌，或公开读取没有统一共享许可。

关系仓库删除 / 反向读取、节点删除或主信息删除接口改变。
已有同功能命名需求入口或并发锁边界形成。
固定安全根路由、自我线程或邮箱已被改成不同的业务分派模型。
完成本切片需要修改核心仓库、概念图服务、语素服务、任务服务或状态服务。
#220 不是最终回归阶段 540、550 已占用，或阶段总成登记 / 冻结接口与假定不一致。
```

出现漂移不得修改代码，退回设计窗口修订 #213-#215。
