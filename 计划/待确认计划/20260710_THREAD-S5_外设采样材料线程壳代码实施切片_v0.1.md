## 7. 实施步骤

### S5-1 排除项复核

复核当前代码中不得出现 D455、体素、真实外设、真实采样 API 接入路径。

### S5-2 外设采样材料线程壳

线程壳只产生模拟或空材料消息。材料必须携带：

```text
来源角色。
发生时间戳。
批次号。
材料摘要。
只读载荷说明。
```

### S5-3 材料入队与满载策略

外部材料队列满时允许丢弃最旧模拟 / 空材料或拒绝生产者；不得生成世界事实。

### S5-4 默认入口自检

```text
THREAD-S5-A1 外设采样材料线程壳可启动和停止。
THREAD-S5-A2 空采样材料可入队。
THREAD-S5-A3 过期模拟材料可丢弃。
THREAD-S5-A4 材料不直接写世界事实。
THREAD-S5-A5 禁止 D455 / 体素 / 真实外设接入。
```

## 8. 失败收口策略

```text
材料队列满载是逻辑内局部失败，不写业务失败事实。
真实外设接入需求出现时停止并登记专项，不得在本切片扩展。
模拟材料被误写入世界事实时必须追根因解决，不得用后续校验兜底。
```
