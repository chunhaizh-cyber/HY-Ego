本轮不存在“诊断到失败就算代码失败”的歧义：A11 首次失败是目标证据，停止后仍须完成两份源码恢复、证据封存和记录提交；构建失败、范围越界、宏关闭仍有差异、源码无法精确恢复、资源未释放或 Git 收口失败才是 R7 执行失败。
