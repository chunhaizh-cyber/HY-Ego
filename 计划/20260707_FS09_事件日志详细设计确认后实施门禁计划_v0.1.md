## 4. 目标服务和目标结构

| 项 | 目标口径 |
| --- | --- |
| 日志系统 | 事件日志作为结构事件审计和后续恢复候选材料 |
| 仓库入口 | 后续只在结构写入确认后记录事件，不让事件日志反向补写结构 |
| 领域服务 | 后续只在领域结构写入完成后记录领域结构事件 |
| 持久化适配 | 后续读取事件日志作为候选材料，不裁决当前事实 |
| 验收入口 | 验证空内容拒绝、未知类别拒绝和写入失败不改结构 |
