## 11. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 无效阶段、状态、概念或上下文 | 逻辑内返回 | 写前拒绝 |
| 已有名称 | 逻辑内返回 | 不生成待命名请求 |
| 同一状态组重复登记 | 正常幂等 | 返回既有登记 |
| 冲突登记或多生命周期目标 | 逻辑内返回 / 结构歧义则追根因 | 不选择第一个 |
| 前置通过后状态创建 / 登记读回失败 | 追根因解决 | 停止初始化依赖路径；未登记、未关联状态不得被解释为生命周期事实 |
| 四根生命周期初始绑定中途失败 | 追根因解决 | 逆序失效本轮已创建关系，生命周期初始化保持未发布 |
| 新概念创建后生命周期关系失败 | 追根因解决 | 清理或隔离本轮新增概念与关系，不发布活动快照 |
| 发布后活动概念生命周期缺失 / 多目标 | 追根因解决 | 停止依赖路径，不声明 S6A 成功 |
