# 6360 子规范 相机外设综合工作流程

编号：6360
日期：20260721
状态：正式子规范

## 1. 定位

本规范只收束双目相机类感知外设从治理需求、任务筹办、外设供料、任务承接、自我侧方法到场景事实提交的综合顺序。各阶段的对象语义、分类、值域、消费权和写入权由对应主题规范唯一规定，不在本规范复制。

直接上级规范：6350

直接依赖规范：5230、6200—6250、6300—6340、6370

相关边界规范：1130—1190、2100、3100、3200、3300

## 2. 综合主链

```text
6320 规定的内部治理或场景维护需要当前事实
-> 形成具名观察、识别、扫描或跟踪需求
-> 5230 规定的任务承接、筹办和方法冻结
-> 6340 规定的外设命令、报告和任务域承接
-> 6350 规定的相机材料成熟与产品交付
-> 当前冻结方法读取本工作项已承接材料
-> 6300、6210、6370 及相邻主题规范规定的自我侧处理
-> 对应领域服务提交特征、状态、动态、存在对应或场景事实
-> 任务域验证实际结果并决定完成、等待或重新筹办
```

任一阶段只能消费上一步按其主题规范正式交付的结构，不得越过相机材料层、任务域承接层或领域提交层。报告、队列项、材料页和观察约束的非事实主体边界服从 6320、6340、6350；本规范不重述其消费权正文。

## 3. 自我侧与外设侧

自我侧与外设侧的权力分层由 6320 定义，相机私有过程和产品交付由 6350 定义，原始与派生深度分账由 6310 定义，场景体素和视觉先验权力由 6250 定义。综合顺序不得用任一可选增强模块替代这些阶段，也不得因模块缺失改变主链事实权力。

## 3.1 成熟度与供给产品二轴

相机材料成熟度轴、逻辑供给产品轴及外显运行模式之间的关系由 6350 唯一定义；物理报告容器、分类视图、来源继承、时效、一次消费和任务绑定由 6340 唯一定义。综合流程只要求二者在“相机产品交付 -> 任务域承接”转换点同时通过，不复制等级、产品或容器枚举。

## 4. 扫描的主动治理位置

扫描需求的治理来源和自我侧分层由 6320 规定；扫描输入、覆盖差集、首次基准和写入边界由 6210 规定；观察事实可用与风险判断分离由 6200 规定。综合流程只把扫描放在“任务域承接完成 -> 冻结扫描方法执行 -> 领域服务提交”这一顺序中，不新增扫描准入或写入规则。

## 5. 四类方法职责

观察像素簇、存在候选及观察、识别、扫描、跟踪的职责边界由 6300 定义，识别唯一性由 6370 定义，扫描写入由 6210、6320 定义。综合流程只保留方法之间不得直连这一顺序约束：一个方法的结构化结果或缺口必须回到需求统一入口、任务承接和新一轮筹办，才能选择后续方法；筹办与无候选边界服从 5230。

## 6. 观察材料到正式存在

```text
6350 裁决相机材料达到可交付条件
-> 6340 裁决报告进入绑定任务的已承接材料
-> 6300 裁决观察像素簇和存在候选
-> 6370 在确定范围内收敛已有存在，或返回空句柄及结构化依据
-> 对应领域入口裁决已确认观察存在及场景关系
```

报告到方法的产品隔离由 6350 规定，报告到任务运行材料的消费隔离由 6340 规定，像素簇和候选不等于正式存在的边界由 6300 规定。综合流程不得把任何外设报告越级当作识别结论、状态变化、新空间事实或场景成员。

## 7. 观察特征优先级和轮廓

观察轮廓、质量和深度分账由 6300、6310 规定，内部结构和体素升级由 6230、6240 规定，识别特征选择、比较值域和反证由 6370 规定。综合流程只按这些规范的交付结果进入下一阶段，不复制特征优先级、编码解释或比较公式。

## 8. 扫描写入边界

扫描的当前值、当前状态、首次基准、未覆盖解释和一阶特征写入边界由 6210、6300、6320 唯一定义。本规范只要求扫描输出经过领域服务提交后才能进入任务结果验证，不复制其计算式或状态规则。

## 9. 目标观察约束

目标观察约束的字段、等待项、调度和相机使用边界由 6340、6350 规定；其非事实主体边界由 6320 规定。综合流程只允许约束随已筹办任务进入外设控制域，并随已承接材料返回冻结方法；不得绕过需求、任务、执行冻结或事实提交。

## 10. 合法来源

材料与质量来源由 6310、6350 规定，特征比较与阈值来源由 6300、6370 规定，状态和动态来源服从对应根规范。综合流程在每个转换点只传递可回查的稳定身份、来源、版本、时间和适用范围，不新增来源类型或默认值。

## 11. 实时性

报告时效、缓存、摘要、丢弃和不可舍弃事件由 6340 唯一定义，相机等待项、采样预算和抢占由 6350 唯一定义。综合流程只要求过期或被舍弃的材料不能继续进入方法和领域提交。

## 12. 禁止规则

1. 不得绕过任一主题规范所拥有的准入、消费、方法或提交阶段。
2. 不得把前一阶段的材料、报告、候选、运行结果或显示投影冒充后一阶段正式事实。
3. 不得因综合顺序引用而改变 6200—6350、6370 各自的唯一语义归属。
4. 执行期发现冻结前应具备的方法结构缺口，按 5230 的筹办漏检规则处理。

## 13. 验证与完成边界

验证按主链逐转换点组合各主题规范的验证合同：需求与筹办读取 5230，观察事实读取 6200—6250，像素簇、质量、分层、外设控制和相机产品读取 6300—6350，识别读取 6370。任一转换点缺少其上一步正式交付或下一步准入证据，综合链均不成立。

没有真实外设材料、任务承接、方法动态、状态变化和领域提交证据时，不得宣称相机综合认知链已经实现。
