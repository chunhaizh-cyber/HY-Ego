# 共享写入未发布身份候选与撤销详细设计

日期：2026-07-13

状态：JY-312 已设计 / #256 待执行 / 不构成 #214 已实现

## 1. 目标

为共享路径下的多步领域创建提供窄失败清理能力：主信息仓库和节点仓库只允许撤销“本次共享令牌创建并仍未发布”的候选身份；已发布身份和普通句柄删除继续只接受独占令牌。

## 2. 当前事实

```text
共享路径令牌可创建主信息、节点和关系。
关系删除令牌入口接受共享路径令牌。
主信息与节点普通删除令牌入口只接受独占令牌。
协调器拒绝同线程嵌套取得许可。
状态 / 动态多步创建必须在发布前失败时清理本次身份。
```

当前 ABI 不存在可满足上述清理的候选能力。

## 3. 数据结构

| 结构 | 用途 | 创建方 | 消费方 | 生命周期 | 是否机器事实 |
| --- | --- | --- | --- | --- | --- |
| `主信息未发布候选` | 证明某主信息由当前仓库、当前完整令牌、本次调用创建且尚未确认 | 主信息仓库 | 创建它的领域服务、主信息仓库确认 / 撤销入口 | 单次调用栈 | 否 |
| `节点未发布候选` | 证明某节点由当前仓库、当前完整令牌、本次调用创建且尚未确认 | 节点仓库 | 创建它的领域服务、节点仓库确认 / 撤销入口 | 单次调用栈 | 否 |
| `核心未发布候选操作状态` | 区分已完成、幂等已撤销、已拒绝和内部不一致 | 核心仓库 | 自检 / 领域组合器 | 局部返回 | 否 |

两类候选不可默认构造、不可复制、允许移动；私有保存仓库实例身份、完整 `结构事务令牌`、精确句柄和局部阶段。候选不序列化、不进入 DTO / 消息 / 日志 / 项目记忆，也不写入仓库记录。

## 4. 接口设计

### 4.1 主信息仓库

```text
创建主信息未发布候选(const 结构事务令牌&) -> optional<主信息未发布候选>
确认未发布候选(主信息未发布候选&, const 结构事务令牌&) -> 核心未发布候选操作状态
撤销未发布候选(主信息未发布候选&, const 结构事务令牌&) -> 核心未发布候选操作状态
```

创建和候选操作验证共享路径令牌。撤销必须同时验证候选所属仓库、完整令牌相等、阶段为未发布、精确记录仍有效且版本一致；随后在主信息仓库本地独占锁内只删除候选记录。普通 `删除主信息` 不改。

### 4.2 节点仓库

```text
创建节点未发布候选(节点类型, 主信息句柄, const 结构事务令牌&) -> optional<节点未发布候选>
确认未发布候选(节点未发布候选&, const 结构事务令牌&) -> 核心未发布候选操作状态
撤销未发布候选(节点未发布候选&, const 结构事务令牌&) -> 核心未发布候选操作状态
```

节点候选撤销只删除本候选节点。调用方必须先按精确句柄删除自己创建的关系；关系清理失败时不得继续撤销节点。节点撤销成功后才能撤销承载主信息候选。普通 `删除节点` 不改。

## 5. 确认与发布

```text
候选创建
-> 领域服务构造全部非发布关系
-> 发布前完整读回
-> 创建唯一最终发布关系
-> 立即确认节点和主信息候选
-> 按最终发布关系公开读回
```

确认只把调用栈候选标为已确认，不修改仓库记录。最终发布关系创建成功后不得再撤销候选；公开读回不一致属于追根因，不得删除已发布结构。

## 6. 失败收口

| 场景 | 分类 | 收口 |
| --- | --- | --- |
| 无效令牌、类型或主信息句柄 | 逻辑内返回 | 第一笔写入前拒绝 |
| 主信息候选创建后节点候选失败 | 追根因解决 | 撤销主信息候选，数量回前态 |
| 非发布关系创建 / 读回失败 | 追根因解决 | 精确删关系 -> 撤销节点候选 -> 撤销主信息候选 |
| 最终发布关系创建失败 | 追根因解决 | 同上；不得确认候选 |
| 错仓库、错令牌、错线程或已确认候选撤销 | 逻辑内拒绝或内部不一致 | 不改变结构；调用方停止依赖 |
| 发布后完整读回不一致 | 追根因解决 | 不撤销、不静默删除、不继续下游 |

候选析构不自动写结构。任何调用方离开作用域前必须显式确认或撤销；自检必须覆盖遗漏检测，但不得依赖析构兜底。

## 7. 并发与权限

```text
候选能力只接受创建时同一完整令牌，因此继续受同线程、同域、同纪元和许可活动记录约束。
共享撤销只处理不可伪造候选，不提供任意句柄删除。
仓库本地锁仍只保护单仓库结构；跨仓库清理顺序由领域服务明确组合。
结构事务接线、协调器、许可类型和活动表不修改。
已发布身份删除、概念安全删除和恢复清理继续使用原独占合同。
```
