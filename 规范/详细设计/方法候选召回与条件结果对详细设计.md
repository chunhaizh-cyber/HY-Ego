## 4. 设计拆分

| 拆分项 | 结构承载 | 第一轮边界 | 不得扩大 |
| --- | --- | --- | --- |
| 方法候选召回材料 | 方法首、方法虚拟存在、条件节点、结果节点、动作入口、稳定动作键索引 | 只读召回候选材料，索引只做入口召回 | 不裁决可执行事实，不排序 |
| 结果能力索引候选 | 方法结果节点、结果场景、后续二次特征或状态材料 | 只登记设计口径，代码前另行确认 | 不把结果文本、缓存命中或 SQL 投影当能力 |
| 条件 / 结果对 | 方法条件节点、方法结果节点、条件结果对角色或后续关系角色 | 第一轮只允许形成候选配对设计 | 不实现条件判定算法 |
| 来源任务关系 | 方法节点到任务节点引用关系 | 只作为来源追溯材料 | 不自动证明方法适配任务 |
| 父 / 前置 / 后续方法 | 方法到方法引用关系及顺序 / 角色约束 | 只作为方法组织材料 | 不实现完整方法图遍历或自动编排 |
| 方法学习材料 | 动态证据、任务结果、需求结算和非权威统计 | 只定义候选材料来源 | 不生成方法成功事实或稳定评分 |

## 5. 服务边界

| 服务 | 允许 | 禁止 |
| --- | --- | --- |
| 方法服务 | 读取方法结构、候选材料、条件 / 结果节点、动作入口和方法关系 | 不直接依赖特征值服务，不裁决需求满足 |
| 任务服务 | 提供来源任务、任务选择方法和任务结果材料 | 不计算候选排序，不替代方法服务关系治理 |
| 需求服务 | 提供目标状态和候选准入材料 | 不写方法关系 |
| 特征 / 状态 / 二次特征服务 | 提供后续条件材料的正式读取入口 | 不被高级服务绕过特征服务访问特征值 |
| 索引仓库 | 只做候选入口召回 | 不裁决业务事实 |
| 显示 / 日志 | 只做人读说明 | 不承载候选、条件、结果或评分事实 |

## 6. 入口拒绝

后续代码实施前必须先拒绝：

```text
无效方法节点。
无效方法首。
方法虚拟存在缺失或不唯一。
方法条件节点角色错误。
方法结果节点角色错误。
条件 / 结果场景缺失、不唯一或目标不是场景。
动作入口缺失、失效或不归属方法首。
稳定动作键为 0 或索引命中目标不属于该方法首。
来源任务目标不是任务节点。
父方法、前置方法或后续方法目标不是方法节点。
父方法自环或明显循环关系。
条件结果对中条件和结果不属于同一方法首。
候选规则未确认却写候选裁决、评分或任务选择方法关系。
日志、显示文本、旧函数名、SQL 或控制面板材料被当作候选依据。
```

拒绝后不得写入方法关系、候选评分、条件结果对、任务选择方法关系或非权威统计。
