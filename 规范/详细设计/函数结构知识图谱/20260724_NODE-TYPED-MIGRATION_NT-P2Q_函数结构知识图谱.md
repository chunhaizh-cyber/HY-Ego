# NT-P2Q 函数结构知识图谱

日期：2026-07-24

合同：`NT-C2/v0.4::P2A-FREEZE/v2`

```text
节点直接身份结构事务域
├─ 运行期域身份_ : uint64，构造时非零且生命周期稳定
├─ 已发布代次_ : uint64，只有真实最后发布后推进
├─ 当前冻结规则版本 : uint32 = 1
├─ 取得统一冻结许可(预期冻结规则版本)
│  └─ 节点直接统一冻结许可（move-only / RAII / 只读）
│     ├─ 有效()
│     └─ 读取见证()
└─ 冻结许可属于本域(许可)

#349 统一冻结协调器
├─ 唯一调用 取得统一冻结许可()
├─ 持有许可覆盖九段读取
├─ 调用 节点直接特征体系数据操作::复制同代次特征值与批次全量(许可)
└─ 完成第 3 / 9 段映射和跨段互证后释放许可

节点直接特征体系数据操作
├─ 绑定同一 节点直接身份结构事务域
├─ 绑定 特征值类型化记录仓
├─ 唯一持有 节点直接特征批次记录仓
└─ 复制同代次特征值与批次全量(许可)
   ├─ 冻结许可属于本域()
   ├─ 读取第一份冻结见证()
   ├─ 特征值类型化记录只读访问器::复制全部已发布记录()
   │  └─ 特征值类型化记录全量读取结果(状态, 结构版本, 记录组)
   ├─ 读取新域全部已发布批次记录()
   ├─ 再读见证 / 两仓结构版本 / 数量
   ├─ 复核记录完整、项目连续、写前可选组和值版本引用
   ├─ 稳定排序两组记录
   └─ 返回 节点直接特征同代次冻结结果

节点直接特征同代次冻结结果
└─ optional<节点直接特征同代次冻结材料>
   ├─ 节点直接统一冻结见证
   ├─ 特征值结构版本 / 数量
   ├─ 批次结构版本 / 数量
   ├─ vector<特征值类型化记录>
   └─ vector<节点直接特征批次冻结记录>

P4 段 9 映射
├─ 批次身份第一分量 = 节点直接特征批次身份命名域 ABI 1
├─ 批次身份第二分量 = 账内批次编号
├─ 账记录格式编码值 = u32 记录格式版本零扩展
├─ 状态 = 已发布
└─ 项目种类 = 写前三可选字段全空 / 全有唯一判定
```

## 边界

- #349 是取得、持有和释放统一冻结许可的唯一算法所有者；#375 不二次取得许可。
- 节点直接新域只读取 `节点直接特征批次记录仓`；旧默认域 4170 账零读取、零拼接。
- 冻结 DTO 归属数据操作模块，不进入 `节点直接领域合同.数据.h`；公开 `节点直接特征服务` 不接触许可。
- 返回材料全部值式自有；许可、仓库、锁、内部容器和参与者均不外泄。
