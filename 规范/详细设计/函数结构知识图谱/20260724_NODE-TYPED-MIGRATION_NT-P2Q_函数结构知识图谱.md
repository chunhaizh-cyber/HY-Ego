# NT-P2Q 函数结构知识图谱

日期：2026-07-24

```text
形成特征值与批次冻结提供者(特征值记录只读访问器, 4170只读访问器)
-> 读取同代次特征值与批次全量(请求)
   -> 取得冻结身份()
   -> 读取全部特征值记录()
   -> 读取全部特征批次发布记录()
   -> 复核冻结身份与数量()
   -> 互证批次引用和值版本()
   -> 排序特征值冻结材料()
   -> 排序特征批次冻结材料()
   -> 返回特征值与批次冻结结果()
```

所有返回为自有值；提供者可移动不可复制。#349 只能调用公开读取入口，不 import P2A 私仓或参与者。
