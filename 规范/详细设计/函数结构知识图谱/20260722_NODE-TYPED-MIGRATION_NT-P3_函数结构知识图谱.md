```text
-> P1 复核节点直接身份和关系
-> P2 值式服务读取所属领域记录
-> 对象专用数据操作组合调用期投影
-> 领域服务返回不可变结果
```

### 7.3 节点退役

```text
对象领域服务::形成对象退役规格
-> P3D::提交节点直接退役
-> 冻结节点 / 当前关系 / 领域记录 / 索引候选

-> 失效关系
-> 各领域记录参与者登记退役
-> 移除索引
-> 标记节点删除
-> 完整读回权威审计
-> 确认并最后发布
-> 投影所有者::使缓存 / 活动图失效并重建
-> P3D::返回分账回执
```

### 7.4 任务筹办、执行、完成与结算

```text
任务服务::取得任务筹办执行权
-> 方法服务::按结果能力召回并回读方法完整结构和生命周期
-> 任务服务::提交方法选择和完整执行冻结
-> 任务服务::结束筹办执行权
-> 任务服务::迁移到排队中并复核冻结
-> 方法服务::形成运行事实并调用动作入口
-> 任务服务::收口执行结果、目标比较、验证、禁止结果和归因
-> 已达成：任务服务::提交任务完成
-> 未达成 / 当前性失效：任务服务::迁移待重筹办
-> 需求服务::在任务完成事实成立后独立提交需求结算
```

第二筹办路径必须在方法召回前因未取得任务占用而返回许可竞争；若两路径同时进入召回或选择，进入 `并发规则` 内部错误。任务筹办直接返回闭合结果，不持久化独立筹办回执。
