本详细设计只定义特征值内容哈希、命中统计和非权威缓存边界，不实现算法，不写 C++。

## 2. 定位

内容哈希和命中统计只用于候选召回、性能优化和人读诊断辅助。它们不是业务事实、不是需求满足依据、不是任务完成依据、不是方法成功依据。

权威复核顺序：

```text
特征值节点身份。
特征值归属目标特征。
Vec 原始值或 I64 原始材料。
容器版本或值版本。
哈希 / 命中统计只作候选和缓存。
```

## 3. 缓存记录结构

| 字段 | 含义 | 第一轮要求 |
| --- | --- | --- |
| 缓存命名空间 | 区分特征值内容哈希、命中观察等缓存 | 必填，未知命名空间拒绝 |
| 来源特征值节点 | 缓存对应的特征值身份 | 必填，必须是有效特征值节点 |
| 来源特征节点 | 特征值归属特征 | 必填，必须与关系复核一致 |
| 来源值类型 | I64、VecI64、VecU64 等 | 必填，必须与原始值容器一致 |
| 来源值版本 | 原始值或容器版本 | 必填，版本不匹配时缓存失效 |
| 内容哈希 | 原始值候选哈希 | 可选，不能单独裁决相等 |
| 命中次数 | 候选命中观察计数 | 可选，不能裁决事实 |
| 最近命中序号 | 命中排序材料 | 可选，只作缓存策略材料 |
| 规则版本 | 哈希或统计规则版本 | 必填，规则变化时缓存失效 |
| 可持久化标记 | 是否允许进入后续快照材料 | 默认否，确认前不得持久化 |

缓存记录不得写入旧特征值主信息字段，不得由显示层或日志层写入。

## 4. 写入方和读取方

| 角色 | 权限 |
| --- | --- |
| 特征服务 | 可在原始值写入后请求刷新内部候选缓存 |
| 特征值服务内部入口 | 可根据原始值生成内容哈希候选材料 |
| 非权威缓存 / 统计入口 | 可记录缓存命名空间、来源版本和命中观察 |
| 需求 / 任务 / 方法服务 | 不得直接读取缓存裁决事实；只能经特征服务读取权威特征状态材料 |
| 显示层 | 只读展示缓存诊断材料 |
| 持久化恢复 | 只能按后续确认边界读取缓存材料，不得恢复为业务事实 |

## 5. 入口前置拒绝

缓存写入或刷新入口必须拒绝：

```text
无效缓存命名空间。
无效特征值节点。
特征值不归属来源特征。
来源值类型未知。
来源值版本缺失。
来源原始值不可读取。
规则版本缺失。
高级服务直接写缓存。
缓存材料试图作为需求 / 任务 / 方法事实写入。
```

缓存读取入口必须：

```text
复核来源特征值当前有效。
复核来源值版本未变化。
复核规则版本匹配。
命中后返回候选，不返回事实结论。
```

## 6. 清空和重建边界

缓存清空必须满足：

```text
不删除特征值节点。
不删除特征到特征值归属关系。
不改变 I64 或 Vec 原始值。
不改变需求、任务、方法、状态、动态、因果引用和结算结构。
```

缓存重建只能从权威结构读取：

```text
特征值节点。
特征归属关系。
I64 或 Vec 原始值。
来源值版本。
```

重建发现冲突时返回候选冲突或失败，不得静默覆盖权威结构。

## 7. 内容哈希边界

内容哈希允许用途：

```text
缩小候选集合。
发现可能重复的特征值材料。
辅助非权威命中统计。
辅助后续持久化审计。
```

内容哈希禁止用途：

```text
直接判断两个特征值相等。
直接判断需求满足。
直接判断任务完成。
直接判断方法成功。
绕过 Vec 原始值比较。
绕过特征服务封口。
```

哈希碰撞必须由原始值复核排除。哈希缺失不得影响权威特征状态读取。

## 8. 命中统计边界

命中次数、最近命中序号和候选热度只表达缓存观察。

```text
命中次数高不代表事实更可靠。
最近命中不代表当前事实成立。
候选热度不代表需求优先级、任务权重或方法成功率。
```

如果后续需要把命中观察转为长期学习材料，必须另建二次特征、状态、动态或因果专项，不得直接复用缓存记录。

## 9. 验收矩阵

| 编号 | 验收项 | 通过标准 |
| --- | --- | --- |
| FS03-CACHE-01 | 命名空间拒绝 | 未知缓存命名空间拒绝 |
| FS03-CACHE-02 | 来源复核 | 无效特征值或非归属特征值拒绝 |
| FS03-CACHE-03 | 版本复核 | 来源值版本变化后缓存失效 |
| FS03-CACHE-04 | 清空不改事实 | 清空缓存后权威特征值仍可读 |
| FS03-CACHE-05 | 哈希不裁决 | 哈希命中仍需原始值复核 |
| FS03-CACHE-06 | 命中统计非权威 | 命中次数不得写成需求、任务、方法事实 |
| FS03-CACHE-07 | 高级服务封口 | 高级服务不得直接读取或写入特征值缓存 |
