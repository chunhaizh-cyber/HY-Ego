## 5. 服务入口设计

### 5.1 既有入口

当前已有入口可作为后续增强基线：

```text
特征服务::创建特征类型()
特征服务::创建特征值(特征节点)
特征服务::写入I64特征值(特征节点, 特征值节点, 值)
特征服务::读取I64特征值(特征节点, 特征值节点)
特征服务::特征值属于特征(特征节点, 特征值节点)
```

这些入口已经表达第一轮封口方向：外部调用方经特征服务访问 I64 特征值，特征值服务不作为高级服务依赖。

### 5.2 待增强入口

后续代码切片可在特征服务中补充以下入口，具体函数名在实施切片中扫描当前头文件后最终确认：

| 设计入口 | 用途 | 返回 | 入口拒绝 |
| --- | --- | --- | --- |
| `读取特征身份` | 按特征句柄复核特征节点身份 | 特征句柄或空 | 无效句柄、错误节点类型、已删除节点 |
| `读取特征语义类型` | 读取特征命名 / 语义类型材料 | 语义材料或空 | 无效特征、关系目标不是允许信息节点、索引命中后类型不符 |
| `创建实例特征槽位` | 在宿主节点下创建或挂接实例特征 | 特征节点或无效句柄 | 无效宿主、宿主类型不允许、无效特征类型、缺关系仓库入口 |
| `读取宿主特征集合` | 读取宿主下特征槽位 | 特征节点集合 | 无效宿主、关系目标类型不符时过滤或返回空 |
| `读取当前I64特征状态材料` | 通过宿主和特征读取当前 I64 值 | I64 或空 | 宿主无该特征、特征值不归属目标特征、未写值 |
| `写入当前I64特征状态材料` | 写入宿主特征当前 I64 值 | 成功 / 失败 | 无效宿主、无效特征、无归属特征值、非归属值、缺前置关系 |

若当前结构无法无歧义表达宿主槽位或唯一当前值，实施切片必须先停在设计确认，不得用旧主信息字段或临时文本绕过。

## 6. 入口前置拒绝

所有写入口至少必须拒绝：

```text
无效特征句柄。
错误节点类型。
无效宿主节点。
宿主节点类型未被本切片允许。
无效特征值节点。
特征值节点不归属目标特征。
缺少特征到特征值归属关系。
缺少宿主到特征槽位关系。
关系仓库或特征值服务内部依赖未注入。
高级服务直接传入或调用特征值服务。
```

拒绝后必须证明：

```text
节点数量不变。
主信息数量不变。
关系数量不变。
索引数量不变。
特征状态材料不变。
```

## 7. 特征值服务封口

对外规则：

```text
特征服务是唯一对外特征值访问入口。
特征值服务只作为特征服务内部依赖。
需求服务、任务服务、方法服务不得直接包含、接收或调用特征值服务。
高级服务需要特征状态材料时，必须通过特征服务。
```

内部规则：

```text
特征值服务只负责创建特征值节点、写入 I64 材料、读取 I64 材料。
特征值服务不判断需求是否满足。
特征值服务不判断任务是否完成。
特征值服务不判断方法是否成功。
特征值服务不处理 VecU、体素、哈希、命中缓存或序列化恢复。
```

## 8. I64 材料边界

I64 在本轮只允许表达第一轮特征状态材料。

允许：

```text
特征服务经内部特征值服务写入 I64 特征状态材料。
需求服务经特征服务读取当前特征状态材料用于结算。
方法服务在任务选择方法后经特征服务修改目标特征状态材料。
动态服务记录改变前值和改变后值材料。
```

禁止：

```text
把 I64 当作需求目标对外语义。
把 I64 当作任务完成事实或方法成功事实。
把 I64 特征值主信息字段扩展成旧项目通用主信息字段。
把 VecU、体素值、稳态列表或近帧候选塞进本轮 I64 入口。
```

## 9. 验收方式

后续代码实施切片至少需要验证：

| 验收项 | 通过标准 |
| --- | --- |
| 特征身份读取 | 有效特征句柄可读回；无效句柄和错误类型返回空。 |
| 索引 / 语义复核 | 索引命中后必须复核节点类型，类型不符不得返回特征事实。 |
| 实例特征槽位 | 有效宿主和特征类型可创建槽位；无效宿主或类型不符拒绝且结构不变。 |
| 当前值读取 | 只有归属目标特征的特征值可读回 I64。 |
| I64 写入 | 写入后必须经特征服务读回同值。 |
| 非归属拒绝 | 非归属特征值写入失败，关系和主信息值材料不变。 |
| 服务封口 | 需求 / 任务 / 方法代码不得直接依赖特征值服务。 |
