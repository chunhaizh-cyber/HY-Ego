## 4. 结构承载

| 业务概念 | 项目承载结构 | 值类型 / 身份结构 | 写入方 | 读取方 | 验证方式 |
| --- | --- | --- | --- | --- | --- |
| 方法身份壳 | `节点类型::方法` 节点 + 主信息锚点 | 方法节点句柄 | 方法服务 `登记方法` | 方法服务、任务服务、显示层只读 | 节点有效且类型为方法 |
| 方法首 | 方法节点 + 方法角色抽象状态 + 方法虚拟存在 | 方法节点句柄、状态节点、存在节点 | 方法服务经状态服务、存在服务写入 | 方法服务、任务服务、方法候选读取方 | 角色状态唯一且为方法首；唯一虚拟存在可读 |
| 方法条件节点 | 方法节点 + 方法角色抽象状态 + 条件场景 + 方法首模板关系 | 方法节点句柄、状态节点、场景节点 | 方法服务经状态服务、场景服务写入 | 方法服务、候选读取方 | 条件角色有效，归属方法首，条件场景唯一 |
| 方法结果节点 | 方法节点 + 方法角色抽象状态 + 结果场景 + 方法首模板关系 | 方法节点句柄、状态节点、场景节点 | 方法服务经状态服务、场景服务写入 | 方法服务、候选读取方 | 结果角色有效，归属方法首，结果场景唯一 |
| 方法角色状态 | 抽象状态节点 + 方法节点模板关系 | I64 方法角色状态值 | 方法服务经状态服务创建 | 方法服务 | 状态值属于方法角色状态枚举，且唯一可读 |
| 方法虚拟存在 | 方法首到存在节点引用关系 | 存在节点句柄 | 方法服务经存在服务创建 | 方法服务、任务服务 | 唯一引用目标为存在节点 |
| 条件 / 结果规格场景 | 条件 / 结果节点到场景节点引用关系 | 场景节点句柄 | 方法服务经场景服务创建 | 方法服务 | 唯一引用目标为场景节点 |
| 动作入口结构锚点 | 动作入口角色方法节点 + 角色状态 + 动作入口状态 + 稳定动作键索引 | 方法节点句柄、状态节点、`uint64` 稳定动作键 | 方法服务经状态服务和索引仓库写入 | 方法服务、后续动作入口流程 | 稳定动作键非 0、不重复，方法首唯一动作入口可读 |

普通 `bool` 在本设计中只能作为局部控制流、入口拒绝或读取可用性返回值，不得持久化为方法可用、方法成功、任务完成或世界事实。

## 5. 请求分流

```text
登记最小方法身份
-> 创建方法主信息和方法节点
-> 返回方法身份壳
-> 不声明方法首、条件、结果或动作入口有效。

创建方法首
-> 登记方法节点
-> 绑定方法首角色状态
-> 经存在服务创建方法虚拟存在
-> 写方法到虚拟存在引用关系
-> 读回有效方法首。

创建方法条件
-> 先复核有效方法首
-> 登记方法节点并绑定方法条件角色状态
-> 经场景服务创建条件规格场景
-> 写条件节点到条件场景引用关系
-> 写方法首到条件节点模板关系
-> 读回有效方法条件节点。

创建方法结果
-> 先复核有效方法首
-> 登记方法节点并绑定方法结果角色状态
-> 经场景服务创建结果规格场景
-> 写结果节点到结果场景引用关系
-> 写方法首到结果节点模板关系
-> 读回有效方法结果节点。

读取方法结构
-> 读取角色状态、虚拟存在、条件集合、结果集合、归属方法首、条件场景和结果场景
-> 每个读取入口都复核节点类型、角色状态和唯一性。

动作入口结构锚点
-> 复核有效方法首、动作入口类型和稳定动作键
-> 拒绝重复动作键和重复动作入口
-> 登记动作入口角色节点并绑定状态
-> 写方法首到动作入口引用关系和稳定动作键索引
-> 只作为后续动作入口 / 方法执行流程门禁。
```

## 6. 入口前置拒绝

后续若进入代码实施，必须在入口先拒绝：

```text
无效方法节点。
节点类型不是方法。
方法首缺角色状态、角色状态不唯一或角色值不是方法首。
方法首缺唯一方法虚拟存在，或引用目标不是存在节点。
创建条件 / 结果时传入的方法首无效。
条件 / 结果节点缺角色状态、角色状态不唯一或角色值错误。
条件 / 结果节点未通过模板关系归属到有效方法首。
条件 / 结果场景缺失、不唯一或目标不是场景节点。
动作入口类型无效。
稳定动作键为 0 或已被索引占用。
方法首已有有效动作入口时重复登记。
动作入口节点等于方法首节点。
方法服务试图直接依赖特征值服务。
旧动作名、函数名、SQL 投影、控制台文本或日志标签被当作动作身份。
```

入口拒绝后不得写方法节点、角色状态、场景关系、动作入口关系、索引记录或统计事实。若入口前置拒绝已经排除错误输入，内部写入结果仍不符合预期，必须按逻辑错误追根因，不能用日志、弹窗、扫描器或显示层兜底修复。

## 7. 服务边界

| 服务 | 本设计允许依赖 | 禁止越界 |
| --- | --- | --- |
| 方法服务 | 创建方法身份、角色状态、条件 / 结果节点、虚拟存在、条件 / 结果场景、动作入口锚点和只读结构材料 | 不召回候选方法，不执行方法，不写任务结果，不直接访问特征值服务 |
| 存在服务 | 创建方法虚拟存在 | 不裁决方法是否可执行 |
| 场景服务 | 创建条件 / 结果规格场景 | 不把规格场景直接当任务运行场景 |
| 状态服务 | 创建方法角色状态和动作入口状态 | 不裁决方法成功或任务完成 |
| 索引仓库 | 绑定稳定动作键到动作入口节点 | 不把动作名文本、函数名或日志标签作为索引事实 |
| 任务服务 | 后续读取方法结构和任务选择方法关系 | 本设计不修改任务关系 |
| 动态服务 / 因果服务 | 后续动作动态和因果证据入口 | 本设计不记录动作动态，不生成因果结论 |

方法服务需要特征状态材料时必须通过特征服务；不得直接包含、接收或调用特征值服务。
