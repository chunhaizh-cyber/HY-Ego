本详细设计只定义值域、复合比较、稳态和区间升格的候选边界，不实现算法，不写 C++。

## 2. 定位

值域、复合比较、稳态和区间升格只能生成候选材料或待复判材料。它们不得直接裁决：

```text
需求满足。
任务完成。
方法成功。
世界事实成立。
稳定因果结论。
```

业务事实仍必须由需求结算记录、任务状态、方法关系、状态 / 动态 / 因果结构或二次特征服务确认后承载。

## 3. 结构分层

| 概念 | 第一轮定位 | 权威性 |
| --- | --- | --- |
| 值域 | 特征值允许范围或候选范围材料 | 候选，不裁决事实 |
| 复合比较 | 多特征或多值材料比较结果 | 候选，不裁决事实 |
| 稳态 | 多次观察后的稳定候选 | 候选，不等于状态事实 |
| 局部区间 | 一段样本范围的候选区间 | 候选，不等于抽象状态 |
| 抽象区间升格 | 从样本候选升为抽象材料的后续过程 | 必须另建二次特征或状态治理切片 |

## 4. 候选记录字段

| 字段 | 含义 | 第一轮要求 |
| --- | --- | --- |
| 候选来源特征 | 候选对应的特征节点 | 必须有效 |
| 候选来源特征值 | 候选对应的特征值节点 | 必须归属来源特征 |
| 候选类型 | 值域、复合比较、稳态、区间升格候选 | 必须明确 |
| 来源原始值类型 | I64、VecI64、VecU64 等 | 必须与特征值材料一致 |
| 来源版本 | 原始值或容器版本 | 必须复核 |
| 候选评分 | Q10000 或后续确认评分 | 可选；不得单独裁决事实 |
| 样本计数 | 形成候选的样本数量 | 可选；不足时只能返回候选不足 |
| 规则版本 | 比较或升格规则版本 | 必填 |
| 复判状态 | 待复判、已复判、拒绝 | 只作候选流程状态 |

候选记录是否持久化必须后续确认。第一轮默认只作为详细设计材料，不新增结构。

## 5. 值域候选边界

值域候选允许：

```text
判断某个特征值是否落入候选范围。
返回候选差距、候选评分或候选不足。
作为方法条件、需求结算或二次特征的输入材料。
```

值域候选禁止：

```text
直接写需求满足。
直接写任务完成。
直接写方法成功。
直接创建抽象状态。
绕过特征服务读取特征值。
```

## 6. 复合比较边界

复合比较必须保留输入材料清单：

```text
比较涉及的特征节点。
比较涉及的特征值节点。
每个值的来源版本。
比较规则版本。
候选评分或差异材料。
```

复合比较结果只能作为候选包，不得静默选择第一个候选为事实。若要进入需求结算、方法条件或二次特征，必须由对应服务复判并写入对应结构。

## 7. 稳态和区间升格边界

稳态候选至少需要：

```text
来源样本集合。
样本计数。
时间或序号范围。
评分或稳定度材料。
规则版本。
来源特征值版本复核。
```

局部区间和抽象区间升格必须另行确认：

```text
升格写入方。
读取方。
是否形成抽象状态、抽象动态或二次特征。
来源样本如何追溯。
失败和撤销如何处理。
```

当前不得实现自动升格，不得把稳态候选直接写成长期基础信息节点。

## 8. 与高级服务的关系

| 服务 | 可用方式 | 禁止 |
| --- | --- | --- |
| 需求服务 | 可经特征服务获取材料，再由需求结算记录复判 | 不得直接调用特征值服务，不得用候选评分直接满足需求 |
| 任务服务 | 可读取需求 / 方法已确认材料 | 不得用候选命中直接完成任务 |
| 方法服务 | 可把候选材料作为条件 / 结果对的输入 | 不得用候选比较直接执行方法成功 |
| 二次特征服务 | 可后续承接候选并形成二次特征 | 未确认前不得自动升格 |
| 因果服务 | 可后续读取动态样本形成因果候选 | 不得把单次比较解释为稳定因果 |

## 9. 验收矩阵

| 编号 | 验收项 | 通过标准 |
| --- | --- | --- |
| FS03-RANGE-01 | 候选不裁决事实 | 值域 / 比较 / 稳态结果不写需求、任务、方法事实 |
| FS03-RANGE-02 | 来源复核 | 候选必须复核特征值归属和来源版本 |
| FS03-RANGE-03 | 候选集合返回 | 多候选必须返回集合或候选包，不静默选第一个 |
| FS03-RANGE-04 | 稳态样本要求 | 稳态候选必须有样本计数和规则版本 |
| FS03-RANGE-05 | 升格门禁 | 抽象区间升格必须另建切片确认 |
| FS03-RANGE-06 | 高级服务封口 | 高级服务不得直接依赖特征值服务 |
| FS03-RANGE-07 | 缺材料不写结构 | 缺值域、缺样本或缺版本时返回候选不足，不写半结构 |
