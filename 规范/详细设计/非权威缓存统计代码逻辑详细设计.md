## 1. 设计定位

本设计承接第 16 项 `流程图/20260708_非权威缓存统计代码逻辑流程图_v0.1.md`，只描述当前统计服务中的结构统计运行期快照、缓存命名空间、缓存清空和非权威边界。

本设计不声明内容哈希、命中观察、需求结算摘要或独立非权威缓存服务已实现。

## 2. 输入入口

```text
节点仓库只读材料
关系仓库只读材料
索引仓库只读材料
缓存命名空间请求
结构统计快照请求
缓存读取 / 清空请求
```

入口必须拒绝无效缓存类型、无效来源类型、无效值类型、缺失失效条件和把缓存用于业务裁决的请求。

## 3. 核心处理

```text
定义缓存命名空间
-> 读取权威结构统计快照
-> 创建统计快照结构
-> 重建结构统计缓存
-> 读取结构统计缓存
-> 判断缓存是否可裁决业务事实
-> 清空结构统计缓存或全部缓存壳
```

当前 `缓存可裁决业务事实` 必须保持 false。

## 4. 结构承载

```text
统计服务：结构统计读取和缓存壳管理
缓存命名空间：缓存类型、来源结构、值类型和失效条件材料
统计快照：有效节点数、有效关系数、有效主键数等运行期只读材料
非权威缓存壳：只服务观察和显示，不写权威事实
```

缓存清空不得改变节点、主信息、关系、索引、需求、任务、方法、状态、动态或结算记录。

## 5. 输出变化

成功路径输出结构统计快照或缓存缺失材料。输出只能供显示层或人工观察读取。

失败路径返回空命名空间、空快照或缓存缺失，不裁决业务事实。

## 6. 当前代码差距

```text
没有独立非权威缓存服务文件。
内容哈希、命中观察、需求结算摘要未落代码。
没有持久化缓存、跨进程恢复或缓存命中统计验收。
没有将特征值哈希或结算摘要接入权威结构后的第二轮实现。
```

## 7. 验收矩阵

| 验收项 | 验证方式 | 通过口径 |
| --- | --- | --- |
| 命名空间准入 | 传入无效缓存命名材料 | 返回空命名空间 |
| 快照读取 | 读取结构统计快照 | 只读取权威结构数量材料 |
| 缓存非权威 | 调用业务裁决判断 | 恒为 false |
| 缓存清空 | 清空结构统计或全部缓存 | 权威结构数量不变 |
| 排除项 | 请求内容哈希或结算摘要实现 | 转后续专项，不在本设计落地 |

## 8. 后续边界

后续若生成施工计划，必须继续保持缓存不得裁决需求满足、任务完成、方法成功或世界事实。
