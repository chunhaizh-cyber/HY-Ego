本详细设计只定义 Vec 原始值容器的结构边界，不实现 C++，不新增旧特征值主信息字段，不迁移旧 VecU / VecI64 函数体。

## 2. 定位

Vec 原始值容器是特征值系统内部的原始材料承载，用于后续表达一组有序数值材料。它只服务特征系统，不作为需求、任务、方法的通用值容器。

第一轮候选类型：

```text
VecI64：有符号 I64 序列。
VecU64：无符号 U64 序列。
```

本设计暂不覆盖：

```text
VecU 旧二进制格式。
体素占据位编码。
三维轮廓融合。
只读指针身份。
内容哈希算法。
序列化恢复实现。
```

## 3. 结构承载

| 结构 | 用途 | 写入方 | 读取方 | 生命周期 | 验证方式 |
| --- | --- | --- | --- | --- | --- |
| `特征值节点` | Vec 容器所属长期身份 | 特征服务经内部特征值服务 | 特征服务 | 随特征值节点生命周期 | 节点类型必须为特征值 |
| `Vec容器记录` | 原始序列材料 | 特征值服务内部入口 | 特征服务、后续哈希候选入口 | 随特征值节点生命周期 | 容器类型、长度和元素可读回 |
| `容器类型` | 区分 VecI64 / VecU64 | 特征值服务内部入口 | 特征服务 | 创建后稳定 | 未知类型拒绝 |
| `元素序列` | 有序原始值 | 特征值服务内部入口 | 特征服务 | 可按确认入口替换或追加 | 长度和元素值复核 |
| `容器版本` | 原始值变化版本 | 特征值服务内部入口 | 哈希 / 缓存候选入口 | 原始值变化时递增 | 哈希候选必须复核版本 |

Vec 容器记录不得作为主信息旧字段迁移。若后续落代码，优先作为特征值服务内部结构或主信息值容器的受控扩展，必须由特征服务对外封口。

## 4. 服务入口候选

后续代码实施切片可考虑以下入口，最终函数名必须在代码实施切片中确认：

```text
写入VecI64特征值(特征节点, 特征值节点, 值序列)
读取VecI64特征值(特征节点, 特征值节点)
写入VecU64特征值(特征节点, 特征值节点, 值序列)
读取VecU64特征值(特征节点, 特征值节点)
特征值Vec容器是否有效(特征节点, 特征值节点)
```

入口只能由特征服务对外提供。特征值服务可有内部入口，但需求、任务、方法、动态、因果、显示层和日志层不得直接依赖特征值服务。

## 5. 入口前置拒绝

Vec 写入入口必须拒绝：

```text
无效特征节点。
错误特征节点类型。
无效特征值节点。
错误特征值节点类型。
特征值不归属目标特征。
容器类型未知。
值序列为空但本入口要求非空。
元素数量超过后续切片确认上限。
VecI64 / VecU64 类型与既有容器类型冲突。
高级服务直接调用特征值服务。
```

读取入口必须拒绝或返回空：

```text
无效特征节点。
无效特征值节点。
特征值不归属目标特征。
容器类型不匹配。
容器记录不存在。
容器版本失效。
```

拒绝后结构不变化至少覆盖：

```text
节点数量不变。
主信息数量不变。
关系数量不变。
索引数量不变。
既有 I64 特征状态材料不变。
既有 Vec 容器记录不变。
```

## 6. 与 I64 第一轮入口的关系

当前已验证的 I64 特征状态材料入口保持不变。Vec 原始值容器不得反向改变 I64 语义。

```text
I64：第一轮单值特征状态材料。
VecI64 / VecU64：后续复合原始值材料。
```

需求目标仍是目标状态，不是 I64 或 Vec。Vec 只能作为特征状态材料或后续候选比较材料，不能直接成为需求目标对外语义。

## 7. 与哈希和缓存的关系

Vec 原始值是内容哈希和命中缓存的权威复核来源。

```text
哈希只能缩小候选范围。
命中统计只能作为非权威缓存材料。
哈希命中后必须复核 Vec 原始值和容器版本。
哈希冲突时以 Vec 原始值比较为准。
```

本设计不实现哈希算法，哈希和缓存边界见 `规范/详细设计/特征值非权威缓存与内容哈希详细设计.md`。

## 8. 与快照恢复的关系

Vec 容器若后续需要持久化，必须进入独立序列化恢复边界专项。第一轮恢复必须遵守：

```text
恢复材料先复核快照格式版本、枚举版本、特征值节点身份和容器版本。
恢复失败不得写半结构。
旧特征值主信息序列化格式不得直接作为新项目格式。
缓存字段不得恢复为业务事实。
```

## 9. 验收矩阵

| 编号 | 验收项 | 通过标准 |
| --- | --- | --- |
| FS03-VEC-01 | Vec 容器身份 | 有效特征值节点可挂接 Vec 容器；无效节点拒绝 |
| FS03-VEC-02 | 归属复核 | 非归属特征值不能写入或读取 Vec |
| FS03-VEC-03 | 类型复核 | VecI64 和 VecU64 类型不混读 |
| FS03-VEC-04 | 原始值读回 | 写入序列后可按同序列读回 |
| FS03-VEC-05 | 拒绝后不变 | 错误输入拒绝后节点、主信息、关系、索引和既有值不变 |
| FS03-VEC-06 | 高级服务封口 | 需求、任务、方法不得直接依赖特征值服务 |
| FS03-VEC-07 | 哈希不裁决 | 哈希命中仍需复核 Vec 原始值 |
