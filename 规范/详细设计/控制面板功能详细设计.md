# 控制面板功能详细设计

更新时间：2026-07-09

状态：已生成 / 可直接生成待确认专项计划 / 未生成施工计划 / 未入队 / 未改 C++ / 未构建 / 未迁移旧函数

## 1. 设计定位

本设计承接 `流程图/20260709_控制面板功能流程图_v0.1.md`，用于把控制面板从旧项目控制面板、SQL 投影、WebView 渲染、D455、体素和外设链路中拆出来，重新定义为海中鱼巣第一轮人读只读视图功能边界。

控制面板第一位功能是人读观察和只读导航，不是机器事实写入方。控制面板不裁决需求满足、任务完成、方法成功、世界事实、状态事实、动态事实或因果结论。

本设计不声明控制面板已接入，不生成代码实施许可，不登记可执行队列，不修改 C++。

## 2. 输入入口

```text
用户明确请求的控制面板功能重建材料
显示服务生成的只读展示材料
世界树 / 概念树 / 语素树 / 需求树 / 任务树 / 方法树只读材料候选
日志审计人读材料
非权威统计快照
控制面板按钮或命令请求材料
数据库 / SQL 审计适配候选请求
WebView / HTML / JSON / 截图渲染候选请求
D455 / 体素 / 外设显示候选请求
```

入口必须拒绝以下请求：

```text
绕过显示服务或领域服务直接读取仓库内部可写结构
通过控制面板按钮直接写节点、主信息、关系、索引、需求、任务、方法、状态、动态或因果结构
通过 HTML / JSON / 截图 / 显示标题 / 日志文本裁决机器事实
通过 SQL / ADO 查询裁决运行期事实
借控制面板接入 WebView、D455、体素或外设能力
```

## 3. 核心处理

```text
接收控制面板功能请求
-> 判断功能类型
-> 只读总览读取显示服务只读材料
-> 树视图 / 详情读取六类树视图只读材料候选
-> 日志 / 统计观察读取人读日志材料和非权威统计快照
-> 操作按钮或命令降级为请求壳
-> 数据库 / SQL 转数据库审计适配候选
-> WebView / HTML / JSON / 截图转人读渲染通道候选
-> D455 / 体素 / 外设转独立专项候选
-> 检查是否越权读取、写事实或文本裁决
-> 组装控制面板只读视图材料
-> 输出人读材料或后续专项门禁
```

控制面板只允许组织人读材料和请求材料。凡是需要改变机器结构的动作，必须转交后续已确认领域服务入口；本设计不执行、不授权、不绕过计划确认门禁。

## 4. 功能分区

| 功能分区 | 第一轮定位 | 允许来源 | 禁止行为 |
| --- | --- | --- | --- |
| 只读总览 | 展示系统当前可读材料摘要 | `显示服务::生成只读展示材料` 后续扩展材料 | 裸读仓库、写事实、裁决任务或需求 |
| 树视图 / 详情 | 展示世界树、概念树、语素树、需求树、任务树、方法树 | 后续领域服务只读材料或只读 DTO | 把状态、动态作为控制面板树视图分类 |
| 日志 / 统计观察 | 展示人读日志和非权威统计 | 日志审计材料、非权威统计快照 | 用日志或统计裁决机器事实 |
| 操作请求壳 | 保存用户操作意图材料 | 操作请求材料 | 按钮直接写节点、关系、任务、方法、状态或动态 |
| 数据库审计候选 | 作为快照、事件日志、恢复、审计或只读查询候选 | 后续数据库审计适配专项 | SQL 裁决运行期事实 |
| 渲染通道候选 | 渲染已生成的人读材料 | 后续 WebView / HTML / JSON / 截图专项 | HTML、JSON、截图承载机器事实 |
| 外设 / 体素候选 | 转独立外设或体素专项 | 后续真实样本和外设材料规范 | 借控制面板提前接入 D455、体素或外设 |

状态和动态不是控制面板树视图分类。状态、动态只能作为领域服务提供的只读材料或人读详情材料被展示，不得在控制面板侧变成新的长期分类、写入口或裁决入口。

## 5. 结构承载

```text
控制面板只读视图材料：人读材料，不承载机器事实。
控制面板树视图材料：领域服务只读输出或后续只读 DTO，不暴露可写内部引用。
控制面板操作请求壳：用户意图材料，不直接写权威结构。
日志观察材料：人读诊断和审计材料，不参与业务裁决。
统计观察材料：非权威统计快照，不参与业务裁决。
数据库审计候选：快照、事件日志、恢复、审计或只读查询材料。
渲染通道候选：HTML / JSON / WebView / 截图的人读输出材料。
```

机器事实仍由节点、主信息、关系、索引、特征、特征值、需求、任务、方法、状态、动态和因果引用等结构承载。控制面板材料不得成为上述结构的替代来源。

## 6. 输出变化

成功路径：

```text
输出控制面板只读视图材料
输出树视图和详情的人读材料
输出日志 / 统计观察的人读材料
输出操作请求壳材料或后续专项门禁
权威结构不变
```

拒绝路径：

```text
越权读取请求被拒绝
写业务事实请求被拒绝
文本裁决请求被拒绝
SQL 裁决运行期事实请求被拒绝
外部显示通道请求进入后续专项门禁
权威结构不变
```

## 9. 验收矩阵

| 验收项 | 验证方式 | 通过口径 |
| --- | --- | --- |
| 服务只读来源 | 构造只读总览请求 | 材料只能来自显示服务或后续领域服务只读入口 |
| 越权读取拒绝 | 构造绕过服务读取请求 | 返回拒绝，权威结构不变 |
| 写事实拒绝 | 构造按钮直接写事实请求 | 返回拒绝，不写节点、主信息、关系、索引或业务结构 |
| 文本裁决拒绝 | 构造 HTML / JSON / 标题 / 日志裁决请求 | 只做人读材料，不生成机器事实 |
| 树视图分类 | 检查控制面板树分类 | 仅包含世界树、概念树、语素树、需求树、任务树、方法树 |
| 状态动态边界 | 检查状态 / 动态展示方式 | 只作为详情材料，不作为控制面板树分类 |
| SQL 边界 | 构造 SQL 裁决请求 | 拒绝裁决运行期事实 |
| 外部通道门禁 | 构造 WebView / 截图 / 外设显示请求 | 转后续专项，不声明已接入 |
| 结构不变 | 对比请求前后权威结构计数或句柄组 | 控制面板输出不改变权威结构 |
