# SQL Server 结构审计投影与控制面板查看详细设计

更新时间：2026-07-10

## 1. 定位

本设计承接用户对本地 SQL Server 数据写入和控制面板查看的明确实施指令。第一轮只持久化 `结构统计快照` 审计投影，不把 SQL Server 作为运行期事实源，不实现核心仓库恢复。

## 2. 输入入口

```text
统计服务读取权威结构统计快照
-> SQL数据库适配.初始化数据库()
-> SQL数据库适配.写入结构统计审计(快照, 来源入口)
-> SQL数据库适配.读取最近结构统计审计(数量上限)
-> 控制面板窗口只读列表显示
```

默认连接目标：

```text
服务器：.\SQLEXPRESS
认证：Windows 集成认证
数据库：海中鱼巣
驱动：优先 ODBC Driver 18 for SQL Server，失败时尝试 ODBC Driver 17 for SQL Server
连接超时：3 秒
```

允许通过外部环境字段 `HY_EGO_SQL_SERVER` 和 `HY_EGO_SQL_DATABASE` 覆盖服务器和数据库名称；环境字段只属于外部配置，不承载机器事实。

## 3. 结构承载

### 3.1 内存材料

`数据库审计记录`：

```text
审计编号
记录时间
来源入口
来源版本
规则版本
节点数
关系数
索引数
```

该结构是数据库查询 DTO，只供适配层和控制面板读取。

### 3.2 SQL 表

表名：`dbo.结构统计审计`

| 字段 | SQL 类型 | 约束 |
| --- | --- | --- |
| 审计编号 | bigint identity | 主键 |
| 记录时间 | datetime2(3) | 默认 `SYSDATETIME()` |
| 来源入口 | nvarchar(128) | 非空 |
| 来源版本 | bigint | 非空且大于 0 |
| 规则版本 | bigint | 非空且大于 0 |
| 节点数 | bigint | 非空且不小于 0 |
| 关系数 | bigint | 非空且不小于 0 |
| 索引数 | bigint | 非空且不小于 0 |

表不保存业务完成状态、需求裁决、任务裁决、方法执行结论或恢复许可。

## 4. 写入边界

1. 只有统计服务生成的有效结构统计快照可以进入适配器。
2. 来源入口为空、来源版本为 0、规则版本为 0或数量溢出时在入口拒绝。
3. 使用参数化 ODBC 写入，不拼接外部来源文本。
4. 数据库写入发生在内存结构已经成立之后；数据库失败不回滚内存事实。
5. 控制面板不调用写入入口，只调用最近记录查询入口。

## 5. 查询与显示边界

控制面板显示最近 50 条结构统计审计记录。列表列为：

```text
编号 / 时间 / 来源 / 节点 / 关系 / 索引
```

刷新按钮同时重读控制面板服务材料和数据库审计记录。查询失败只在数据库状态栏和人读列表状态中显示，不关闭窗口、不改核心仓库。

## 6. 失败收口

| 阶段 | 分类 | 处理 |
| --- | --- | --- |
| SQL Server 不可连接 | 逻辑内返回 | 返回数据库不可用材料，核心运行继续 |
| 数据库或表尚不存在 | 允许分支 | 初始化数据库和表后继续 |
| 合法材料进入写入后执行失败 | 追根因解决 | 返回精确 ODBC 诊断，不宣称写入成功 |
| 写入成功但读回不匹配 | 追根因解决 | 返回读回失败，不改变内存事实 |
| 控制面板查询失败 | 逻辑内返回 | 显示不可用原因，其他只读材料继续展示 |

## 7. 明确排除

```text
不修改节点、主信息、关系、索引仓库。
不从数据库恢复或覆盖运行期仓库。
不实现节点、主信息、关系、索引明细序列化。
不迁移旧鱼巢 SQL 表或旧 ADO 函数体。
不引入 WebView、HTTP、本机服务端口、D455、体素或外设。
不让数据库记录裁决需求、任务、方法、状态、动态或因果。
```

## 8. 验收矩阵

| 编号 | 验收项 | 通过标准 |
| --- | --- | --- |
| DB-PANEL-A1 | 本地实例连接 | Windows 集成认证连接 `.\SQLEXPRESS` 成功 |
| DB-PANEL-A2 | 数据库初始化 | `海中鱼巣` 数据库存在 |
| DB-PANEL-A3 | 表初始化 | `dbo.结构统计审计` 存在且字段完整 |
| DB-PANEL-A4 | 参数化写入 | 有效结构统计快照追加成功 |
| DB-PANEL-A5 | 写后读回 | 最近记录数量与本次快照一致 |
| DB-PANEL-A6 | 控制面板查看 | 原生窗口显示数据库审计列表 |
| DB-PANEL-A7 | 刷新查询 | 刷新只重读材料，不写核心仓库 |
| DB-PANEL-A8 | 数据库非权威 | SQL 结果不参与程序通过、业务裁决或仓库恢复 |
| DB-PANEL-A9 | 故障隔离 | 数据库不可用时核心结构和窗口其他材料可继续 |
| DB-PANEL-A10 | 排除项 | 无旧表迁移、WebView、HTTP、D455、体素或外设接入 |

## 9. 完成声明边界

完成后只能声明：

```text
SQL Server 结构统计审计投影可以初始化、追加和读回。
控制面板可以只读查看最近审计记录。
数据库失败不会改变运行期内存事实。
```

不得声明完整仓库持久化、完整数据库读写系统、数据库恢复、旧数据库迁移或控制面板业务操作完成。
