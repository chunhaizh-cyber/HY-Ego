本详细设计只定义抽象状态 / 抽象动态候选、升格、衰减和淘汰的结构边界，不实现抽象学习、聚类、统计或淘汰算法。

## 2. 定位

抽象状态和抽象动态是长期基础信息节点。实例状态和实例动态是场景内临时零散节点，不能直接作为长期语素入口对应信息。

抽象治理的第一轮目标：

```text
定义候选来自哪里。
定义候选不能直接裁决什么。
定义升格前必须复核哪些材料。
定义衰减和淘汰只能如何表达。
定义后续代码切片的拒绝和验收边界。
```

## 3. 候选材料

| 候选材料 | 来源服务 | 第一轮准入 |
| --- | --- | --- |
| 实例状态材料 | 状态服务 | 主体、场景、值、发生时间戳完整 |
| 实例动态材料 | 动态服务 | 主体、场景、目标、前后值、发生时间戳完整 |
| 二次特征组成材料 | 二次特征服务 | 组成项可追溯到状态或动态 |
| 动态聚合候选 | 动态服务后续候选入口 | 只作候选材料 |
| 因果候选 | 因果服务后续候选入口 | 只作证据材料 |
| 语素名称材料 | 语素服务 | 只能绑定已生成的长期抽象节点 |

禁止候选材料：

```text
日志文本。
控制台输出。
SQL 行。
控制面板显示状态。
D455、体素或外设材料。
线程事件。
自然语言说明。
```

## 4. 抽象状态治理

抽象状态候选允许来自多个实例状态的相似状态材料，但候选本身不是长期结论。

升格为抽象状态前必须满足：

```text
来源实例状态均有效。
来源实例状态材料可读。
状态值或状态分类材料可解释。
实例发生时间戳不进入抽象状态本体。
抽象状态写入由状态服务完成。
语素服务只绑定已存在抽象状态。
```

抽象状态禁止承载：

```text
单次实例发生时间。
显示名称裁决。
任务完成事实。
需求满足事实。
方法成功事实。
日志统计事实。
```

## 5. 抽象动态治理

抽象动态候选允许来自实例动态、动态聚合候选和因果候选材料，但候选本身不是动作、方法或稳定因果结论。

升格为抽象动态前必须满足：

```text
来源实例动态均有效。
来源动态材料具备主体、场景、目标、前值、后值和发生时间戳。
来源动作为空或为动作入口角色节点。
来源方法如存在必须由方法服务复核。
实例动态单次时间不进入抽象动态本体。
抽象动态写入由动态服务完成。
```

抽象动态禁止承载：

```text
线程动作来源。
函数名动作来源。
控制台命令。
外设动作事实。
本能动作执行事实。
稳定因果结论。
```

## 6. 评分、衰减和淘汰边界

第一轮只允许把评分、命中、衰减和淘汰作为后续候选治理口径，不作为已实现结构。

若后续需要落结构，必须另行设计：

```text
候选评分 Q10000。
候选版本。
证据样本计数。
最近复核时间。
淘汰原因结构。
旧候选失效方式。
```

禁止：

```text
用 bool 表示抽象是否成立。
用日志表示淘汰原因。
用显示文字表示评分。
用缓存命中表示业务事实。
用单个轻量因果引用表示稳定结论。
```

## 7. 入口拒绝矩阵

| 编号 | 拒绝项 | 处理 |
| --- | --- | --- |
| BASE-ABSTRACT-01 | 候选来源为空 | 返回空候选，不写抽象节点 |
| BASE-ABSTRACT-02 | 来源实例状态 / 动态无效 | 拒绝该候选 |
| BASE-ABSTRACT-03 | 来源材料缺主体、场景、值、目标、前后值或时间戳 | 返回候选不足 |
| BASE-ABSTRACT-04 | 候选要求把实例时间写入抽象本体 | 拒绝升格 |
| BASE-ABSTRACT-05 | 候选要求写任务、需求或方法事实 | 拒绝越权写入 |
| BASE-ABSTRACT-06 | 候选材料来自 SQL、控制面板、日志、D455、体素、外设或线程 | 拒绝外部材料 |
| BASE-ABSTRACT-07 | 候选要求自动淘汰长期抽象节点 | 拒绝，需独立生命周期切片 |

拒绝后不得新增抽象状态、抽象动态、关系、索引、需求、任务、方法或因果事实。

## 8. 验收矩阵

| 编号 | 验收项 | 通过标准 |
| --- | --- | --- |
| BASE-ABSTRACT-V01 | 候选只读 | 候选读取不写长期抽象节点 |
| BASE-ABSTRACT-V02 | 升格前复核 | 升格前来源材料完整且可追溯 |
| BASE-ABSTRACT-V03 | 实例时间不入本体 | 抽象状态 / 动态本体不携带发生时间戳 |
| BASE-ABSTRACT-V04 | 语素绑定边界 | 语素只能绑定已存在长期抽象节点 |
| BASE-ABSTRACT-V05 | 评分非权威 | 评分、命中、衰减和淘汰候选不裁决业务事实 |
| BASE-ABSTRACT-V06 | 外部材料排除 | SQL、控制面板、D455、体素、外设、线程和日志材料被拒绝 |
