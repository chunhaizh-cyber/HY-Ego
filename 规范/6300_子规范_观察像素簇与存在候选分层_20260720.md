# 6300 子规范 观察像素簇与存在候选分层

编号：6300
日期：20260721
状态：正式子规范

## 1. 定位

本规范规定外设像素材料、观察像素簇、存在候选、已确认观察存在以及观察、识别、扫描、跟踪四类方法之间的事实层级和写入边界。

直接上级规范：1130、1140

直接依赖规范：6200、6340、6350

相关边界规范：1150、1160、1170、1180、2100、6210—6250

## 2. 观察材料分层

```text
原始帧、像素、点云和单帧分割结果
-> 经质量门、跨帧稳定和范围组织的观察像素簇
-> 自我侧引用观察像素簇形成存在候选
-> 自我验证链确认后形成已确认观察存在
```

原始帧和单帧簇只属于外设内部材料、诊断和回查层。单帧质量良好不等于跨帧稳定，也不得交付为已确认观察事实。

存在候选必须通过稳定句柄引用来源观察像素簇、观察材料集和范围，不得复制整套像素事实。存在候选、冲突对应和证据不足都是过程状态；只有自我验证链可以确认观察存在。

## 3. 可观测单位对应

当前观察范围内的有效可观测单位至少区分：

- 稳定对应已有存在；
- 稳定对应新存在候选；
- 多重对应冲突；
- 证据不足、尚未稳定对应。

无效深度、材料缺失、不可解引用、遮挡和质量不足先作为不可观测或无效材料排除，不进入有效单位完成计数。

“可观测单位存在对应闭合”必须同时满足：

```text
已稳定对应存在的有效单位数量 = 有效可观测单位数量
未稳定对应单位数量 = 0
多重对应冲突单位数量 = 0
```

闭合只说明限定观察材料集内的有效单位已有稳定归属，不表示世界对象全部确认、全场景特征明确、风险明确或目标始终可见。

自我认知中不存在“背景”这一存在类别。视觉工程中的背景必须转写为已归属的静止存在、环境存在、低动态存在或暂不高频处理区域；尚未归属的区域仍是待识别、待复验或低质量材料。

## 4. 观察像素簇的结构材料

观察像素簇至少关联：

- 来源外设和观察材料集；
- 观察范围、ROI、坐标系和来源时间；
- 簇身份及簇掩码；
- 平面轮廓、质量、稳定性和不可观测原因；
- 彩色局部图、深度局部图或其它回查证据的受控引用；
- 当前对应状态和候选存在引用。

局部图只作观察证据、人工复核、递归细分和跨轮比较，不是权威真值。掩码参与簇生成、子轮廓提取、像素归属、三维建模和对应统计，但不得塞入普通 I64 数值槽。

## 5. 轮廓和值解释

读取平面轮廓或空间极值轮廓时，必须先由特征类型确定语义，再解码底层向量值。平面轮廓按二维坐标链解释；空间极值轮廓按三维坐标链解释。坐标链长度必须满足维数整除，坐标系、ROI、来源材料和版本由伴随关系或材料承载。

底层向量不得直接参加 I64 区间比较。识别、扫描和跟踪需要比较时，必须先由合法规则或方法生成中心偏差、面积差异、周长差异、长宽比差异、掩码重合率、轮廓点平均误差、轮廓点最大误差或综合匹配评分等二次特征。

## 6. 四类方法的写入边界

| 方法 | 合法输入 | 合法输出 | 禁止事项 |
| --- | --- | --- | --- |
| 观察 | 合法外设材料、观察范围和质量证据 | 新观察像素簇、存在候选材料，并可向存在特征集合追加合法一阶观察特征 | 不得直接确认世界存在 |
| 识别 | 已有特征、已确定范围和候选存在集合 | 已有存在稳定句柄或空句柄；过程缺口另行结构化承载 | 不得补写一阶特征，不得创建存在 |
| 扫描 | 自我内部场景、已确认存在、扫描关注集合、合法外设数据和可比基准 | 更新当前特征值或当前状态；输出变化、未覆盖解释或空信息 | 不得从原始材料生产新一阶特征，不得直接创建存在 |
| 跟踪 | 已确认目标存在、唯一凭证特征、持续观察窗口和范围 | 指定目标的新空间坐标、跟踪状态或空信息 | 不得替代全场扫描或把局部稳定扩大为存在确认 |

第一次扫描没有历史可比材料时，只能建立当前特征值和状态基准，不得输出“发生变化”。扫描发现新区域时只能形成未稳定对应特征或需重识别缺口。

第一版跟踪目标必须是已确认存在。没有目标稳定句柄、目标特征和持续观察窗口时不得启动跟踪。

## 7. 外设供给与观察结果隔离

相机材料成熟度、逻辑供给产品及其相互隔离由 6350 唯一定义；通用报告、物理队列、一次消费、任务绑定和任务域承接权由 6340 唯一定义。本规范不重复产品枚举、成熟度等级或队列实现规则，只规定进入观察对象层后的转换：

```text
符合 6350 观察交付合同且已经按 6340 绑定任务的材料引用
-> 观察方法按本规范形成观察像素簇
-> 自我侧引用观察像素簇形成存在候选
-> 合法领域入口裁决存在对应或正式事实
```

外设供给只提供方法输入材料，不能直接满足世界事实或任务目标。存在候选、对应摘要、变化事实、局部细分和已确认观察存在必须由自我侧方法及合法领域入口形成。识别方法只读取观察链已经形成的已有特征和候选范围，不把外设报告直接当作存在结论。

观察、识别、扫描和跟踪不得直接互相调用。衔接只能沿“结构化方法结果或缺口 -> 需求统一入口 -> 任务承接和筹办 -> 方法选择”。

## 8. 线程和事实权力

外设线程可以持续采集、校正、滤波、分割、跟踪、裁剪局部图并调度报告，但线程不是动作来源，也不是世界事实写入者。需要形成动作动态、状态、特征值或存在关系时，必须由明确方法和领域提交入口完成。

长期身份使用节点稳定句柄、关系和索引。外设帧号、簇号、外设内部轨迹号和任务号只作回查材料，不得成为世界存在身份。

## 9. 验证与完成边界

验证至少覆盖材料层级、对应计数、无背景归属、轮廓类型解释、四方法写入边界、首次扫描、局部不扩大、报告隔离以及线程不写事实。

缺少真实连续外设材料、自我侧正式承接、存在确认和状态动态证据时，不得宣称识别、扫描、跟踪或当前范围对应闭环已实现。
