# 4130 子规范 特征值三态比较底层逻辑

编号：4130
日期：20260720
状态：正式子规范

## 1. 定位

三态比较是两个同域、可比较特征值形成方向关系的底层协议，不是持久化业务对象。

上级规范：1175、2100。

## 2. 比较结果

比较函数返回有符号比较量 `r = compare(左值, 右值)`：

```text
r < 0  左值小于右值
r = 0  左值等于右值
r > 0  左值大于右值
```

底层逻辑只依赖符号，不依赖绝对幅度。实现可返回安全差值，但幅度只供量化、排序或变化量计算，不能改变三态语义。

不可比较不是第四种结果。类型、维度、单位、版本、值域或目标缺失必须在比较前拒绝或进入内部错误。

## 3. 左右值合同

每个关系生成点必须固定左右语义并公开声明。例如状态迁移统一比较旧值与新值，目标判断统一比较实际值与目标值。任何模块不得私自反转后继续复用同一方向含义。

指针、枚举、时间和复合材料必须先进入各自正式比较域；地址和文本不直接参加业务比较。

## 4. 比较与命中分离

比较器只产生方向。允许方向、比较模板和条件随后判断该方向是否命中：

```text
比较(左值, 右值)
-> 提取符号
-> 与模板允许方向匹配
-> 形成具名条件裁决
```

大于等于、小于等于、不等于和方向集合是三种原子方向的组合，不新增第四类比较结果。

方向区间描述变化幅度域，允许方向描述符号是否命中，二者不得互换。

## 5. 使用位置

三态比较用于方向关系、排序、状态迁移、方法条件、二次特征、因果样本、需求目标和结算前事实比较。只保存对象或单个原始值时不调用比较器。

“促进、抑制、提升、超时、满足”等解释词只能由具体比较对象、符号和规则临时解释，不得成为底层字段或直接结算依据。

## 6. 禁止事项

- 禁止比较器返回未知/不可比作为业务方向。
- 禁止把命中 bool 当成三态比较结果。
- 禁止保存方向掩码作为原始状态。
- 禁止用方向词替代左右值、单位和版本证据。
- 禁止由比较量有推进直接宣称目标完成。
