# 1195 根规范 因果模板

编号：1195
日期：20260720
状态：正式根规范

## 1. 根定位

因果模板是因果对象的抽象用途角色，不是独立于因果本体的平行存储类型。

## 2. 本项目承载

因果模板复用 1190 规定的稳定因果身份、原因结果结构、证据、适用锚点、基础计数和版本。模板角色通过正式模板关系、适用范围或因果完整结构键表达；不得为模板另建自由业务字段。

当前因果查询候选和因果模式结构键只可作为模板召回或形成候选的材料，未发布前不构成模板事实。

## 3. 事实与边界

1. 因果模板不得创建独立本体字段或独立主信息形态。
2. 不得绕过 1190 为模板增加解释分、成功率画像或自由文本原因。
3. 模板用途只能由因果本体的原因结果角色、适用锚点、证据、计数和版本解释。
4. 模板召回命中不等于因果适用于当前任务；仍须复核当前条件、范围和版本。

## 4. 写入、读取与验证

模板关系只能由因果领域服务在完整因果事实之上建立。有效模板必须能回读同一因果本体、合法模板角色、适用范围和当前版本；删除模板角色不得删除因果本体或证据。

## 5. 子规范路由

因果模板的召回、补完、适用范围和目标投影分别进入 4310、4320；所有条款继续服从 1190。
