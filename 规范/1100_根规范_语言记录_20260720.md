# 1100 根规范 语言记录

编号：1100
日期：20260722
状态：正式根规范

## 1. 根定位

语言记录是保存外来或内部语言材料、结构位置、解析版本和入口关系的语言域对象。它是材料与解析锚点，不是世界事实、需求、任务、方法或因果结论。

## 2. 本项目承载

语言记录使用含直接稳定主键的语言对象节点、语言域类型化原文 / 位置 / 解析版本材料、组成 / 归属 / 顺序关系和索引共同承载。不得因缺少专用语言对象角色而用语素节点或基础信息节点冒充；专用角色仍必须复用节点、稳定句柄、关系和索引架构。

## 3. 事实边界

语言记录可以承载原始文本材料、语言类型、位置、组成关系、解析候选和来源时间。文本、词性和解析标签都只是语言材料，不直接成为业务语义权威。

语言材料触发业务逻辑时，必须经解析、语素入口或对应领域提交入口，转换为有稳定身份和正式关系证据的项目对象。

## 4. 写入、当前性与禁止

语言领域服务是语言记录和组成关系的写入方；其它领域只读取或提交解析结果。解析更新必须保留来源版本、当前发布和历史版本。

禁止把原文、标题、句子、日志或解析标签直接作为存在身份、状态、需求、任务、方法、因果或结算事实。

## 5. 子规范路由

2300—2330 分别规定文章、段落、自然句和子句的结构层级；2200、2210 规定词与语素入口。
