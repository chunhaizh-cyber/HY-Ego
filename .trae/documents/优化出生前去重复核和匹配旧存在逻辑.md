# 优化存在匹配逻辑：删除去重复核，简化匹配流程

## 问题分析
当前的存在匹配逻辑存在以下问题：

1. **去重复核模块复杂**：增加了代码复杂度，逻辑不够清晰
2. **匹配标准不明确**：没有明确的匹配优先级和标准
3. **空间占位匹配不足**：未充分利用空间占位信息进行匹配

## 优化方案

### 1. 删除去重复核模块
- **简化逻辑**：直接删除 `检查候产房出生` 函数中的去重复核部分
- **明确流程**：候产房产出后，直接创建新存在或匹配旧存在

### 2. 优化旧存在匹配逻辑
- **匹配优先级**：
  1. **跟踪ID匹配**（最高优先级）：未丢失跟踪时，跟踪ID相同则匹配
  2. **空间占位匹配**：空间占位相同或叠加为唯一匹配标准
  3. **多特征匹配**：轮廓、子轮廓、颜色等特征匹配（兜底）

### 3. 实现空间占位匹配
- **AABB相交检测**：检查新存在与现有存在的空间占位是否相同或叠加
- **匹配标准**：空间占位相同或叠加时，视为同一存在

// 检查空间占位匹配
bool 检查空间占位匹配(const 结构体_存在观测& obs, const Vector3D& candPos, const Vector3D& candSize) const noexcept {
    // 计算观测的AABB
    const double obsHalfX = obs.尺寸.x / 2.0;
    const double obsHalfY = obs.尺寸.y / 2.0;
    const double obsHalfZ = obs.尺寸.z / 2.0;
    const double obsMinX = obs.中心坐标.x - obsHalfX;
    const double obsMaxX = obs.中心坐标.x + obsHalfX;
    const double obsMinY = obs.中心坐标.y - obsHalfY;
    const double obsMaxY = obs.中心坐标.y + obsHalfY;
    const double obsMinZ = obs.中心坐标.z - obsHalfZ;
    const double obsMaxZ = obs.中心坐标.z + obsHalfZ;
    
    // 计算候选存在的AABB
    const double candHalfX = candSize.x / 2.0;
    const double candHalfY = candSize.y / 2.0;
    const double candHalfZ = candSize.z / 2.0;
    const double candMinX = candPos.x - candHalfX;
    const double candMaxX = candPos.x + candHalfX;
    const double candMinY = candPos.y - candHalfY;
    const double candMaxY = candPos.y + candHalfY;
    const double candMinZ = candPos.z - candHalfZ;
    const double candMaxZ = candPos.z + candHalfZ;
    
    // 检查AABB是否相交
    bool intersects = 
        (obsMinX < candMaxX && obsMaxX > candMinX) &&
        (obsMinY < candMaxY && obsMaxY > candMinY) &&
        (obsMinZ < candMaxZ && obsMaxZ > candMinZ);
    
    return intersects;
}

// 计算匹配得分
double 计算匹配得分(const 结构体_存在观测& obs1, const 结构体_存在观测& obs2) const noexcept {
    double totalScore = 0.0;
    double maxPossibleScore = 0.0;
    
    // 轮廓相似度
    const double contourWeight = 3.0;
    if (!obs1.轮廓编码.empty() && !obs2.轮廓编码.empty()) {
        float contourSim = 通用函数类::轮廓相似度_VecIU64(obs1.轮廓编码, obs2.轮廓编码);
        totalScore += contourSim * contourWeight;
        maxPossibleScore += contourWeight;
    }
    
    // 颜色相似度
    const double colorWeight = 2.0;
    if (obs1.平均颜色可信 && obs2.平均颜色可信) {
        // 计算颜色距离
        const float r1 = obs1.平均颜色.r / 255.0f;
        const float g1 = obs1.平均颜色.g / 255.0f;
        const float b1 = obs1.平均颜色.b / 255.0f;
        const float r2 = obs2.平均颜色.r / 255.0f;
        const float g2 = obs2.平均颜色.g / 255.0f;
        const float b2 = obs2.平均颜色.b / 255.0f;
        
        const float colorDist = std::sqrt(
            std::pow(r1 - r2, 2) +
            std::pow(g1 - g2, 2) +
            std::pow(b1 - b2, 2)
        );
        
        totalScore += (1.0f - colorDist) * colorWeight;
        maxPossibleScore += colorWeight;
    }
    
    // 位置相似度
    const double positionWeight = 2.0;
    const double dist = std::sqrt(
        std::pow(obs1.中心坐标.x - obs2.中心坐标.x, 2) +
        std::pow(obs1.中心坐标.y - obs2.中心坐标.y, 2) +
        std::pow(obs1.中心坐标.z - obs2.中心坐标.z, 2)
    );
    
    // 距离越近得分越高
    const double maxDist = 200.0; // 最大距离阈值
    double positionScore = std::max(0.0, 1.0 - (dist / maxDist)) * positionWeight;
    totalScore += positionScore;
    maxPossibleScore += positionWeight;
    
    // 尺寸相似度
    const double sizeWeight = 2.0;
    const double sizeRatioX = std::max(obs1.尺寸.x, obs2.尺寸.x) / std::min(obs1.尺寸.x, obs2.尺寸.x);
    const double sizeRatioY = std::max(obs1.尺寸.y, obs2.尺寸.y) / std::min(obs1.尺寸.y, obs2.尺寸.y);
    const double sizeRatioZ = std::max(obs1.尺寸.z, obs2.尺寸.z) / std::min(obs1.尺寸.z, obs2.尺寸.z);
    
    double sizeScoreX = std::max(0.0, 1.0 - std::abs(sizeRatioX - 1.0));
    double sizeScoreY = std::max(0.0, 1.0 - std::abs(sizeRatioY - 1.0));
    double sizeScoreZ = std::max(0.0, 1.0 - std::abs(sizeRatioZ - 1.0));
    double averageSizeScore = (sizeScoreX + sizeScoreY + sizeScoreZ) / 3.0;
    totalScore += averageSizeScore * sizeWeight;
    maxPossibleScore += sizeWeight;
    
    // 计算最终得分
    return maxPossibleScore > 0 ? totalScore / maxPossibleScore : 0.0;
}
