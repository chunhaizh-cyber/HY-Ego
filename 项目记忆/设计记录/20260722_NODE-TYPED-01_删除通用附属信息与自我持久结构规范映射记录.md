## 1. 用户裁决

本轮删除现有通用附属信息体系，不保留其编号、句柄、记录、仓库、无类型值容器、节点绑定和拓扑锚点，也不得换名后继续保留同一种通用容器。

新的唯一分账为：

```text
对象身份 -> 节点直接字段和稳定句柄

关系无法表达的必要原始值 -> 所属领域类型化数据记录
定位加速 -> 可重建索引
完整读取材料和列表 -> 非持久组合读取投影
```

## 5. 自我和恢复反例

必须拒绝或判为内部结构错误：

- 节点有效但稳定主键为空、重复或复义；
- 自我不存在、出现两个当前自我、没有内部世界或具有两个当前内部世界；
- 自我是自身内部世界成员，或子存在只有数组而无成员关系；
- 自我实例槽缺宿主、模板或当前值关系；
- 状态 / 动态列表与主体关系查询不一致；
- 节点、关系和领域记录分别可读但全量互证失败；
- 恢复材料缺失失效关系、历史事实或必需领域记录；
- 只恢复索引、缓存、列表或投影而缺权威结构；
- 恢复失败后改变当前运行期上下文。
