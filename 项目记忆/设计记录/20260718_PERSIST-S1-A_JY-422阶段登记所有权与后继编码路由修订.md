## 2. 裁决

1. #304—#307 均原地升级为正文 v0.2，保持队列号、阶段号、依赖和领域范围不变。
2. 四个计划分别允许 `入口.cpp` 只做 Debug import、一个结果槽、一次阶段登记和最终具名汇总。
3. 通用 `自检.运行器.ixx` 改列禁止文件；入口生产分支、旧兼容正文和专项自检正文继续禁止修改。

5. PERSIST-S1-A 是严格串行链，入口文件由相邻计划顺序接管，不需要新增并行汇合计划。
6. 恢复总图同步为 `#298 -> #299 / #302 -> #301 -> #309 -> #303 -> #304—#307`。
7. #226 升为正文 v0.4，只继承完成后的九段 DTO 和入口接线边界；仍无代码许可，#307 后必须按实际接口再次修订。
