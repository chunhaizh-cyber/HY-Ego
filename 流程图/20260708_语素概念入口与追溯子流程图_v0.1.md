# 2.4 语素概念入口与追溯子流程图

更新时间：2026-07-08

## 依据

```text
海中鱼巣/领域/语素服务.h
海中鱼巣/入口.cpp
```

## 说明

本子流程表达 `语素服务::创建概念入口` 的代码逻辑。它组合文本准入、概念节点准入、语素入口创建和概念追溯追加。

## 流程图

```mermaid
flowchart TD
    A["输入：文本 / 对应信息节点 / 概念节点 / 可选主键"] --> B["2.1 文本准入"]
    B --> C{"文本是否可作为最小词单元"}
    C -- "否" --> R1["拒绝：返回无效句柄"]
    R1 --> X["结构不变化"]
    C -- "是" --> D["2.2 概念节点准入"]
    D --> E{"概念节点是否可绑定"}
    E -- "否" --> R2["拒绝：返回无效句柄"]
    R2 --> X
    E -- "是" --> F["2.3 创建语素入口"]
    F --> G{"语素入口是否有效"}
    G -- "否" --> R3["返回无效句柄"]
    R3 --> X
    G -- "是" --> H["追加语素概念追溯"]
    H --> I{"追溯关系是否已存在"}
    I -- "是" --> J["返回入口句柄：不重复写关系"]
    I -- "否" --> K["创建语素概念追溯关系"]
    K --> J
```

## 关键边界

```text
文本不合法时不能创建入口。
概念节点不合法时不能创建入口。
创建语素入口失败后不能继续追加概念追溯。
概念追溯是关系材料，不是稳定概念裁决或自然语言理解完成。
```
