NODE-TYPED-MIGRATION NT-P2A 特征值类型化记录迁移施工流程图

身份：NT-P2A 施工目标图；依赖 NT-P1 预冻结待实现接口合同复核、叶子计划登记和正式派发；真实接线和完整验证由 #352 完成。

权威分账：类型化记录独占原始值，关系独占拓扑，4170 独占批次幂等审计，组合投影不持久化。

禁止：主信息值槽、无类型通用容器、关系端点副本、批次账复制原始值和发布前半结构可见。

flowchart TD
    A["调用语境：节点直接特征服务收到创建槽位、发布初始值、换代值或特征批次请求"] --> B["节点直接特征服务解释值域、合法来源、幂等身份和业务规则；外部及高级服务不得直接访问特征值记录"]
    B --> C{"请求身份、宿主、定义、来源、原始类型、值域、格式版本和预期当前证据是否完整"}
    C -->|"否"| LR0["逻辑内返回：具名入口拒绝，节点、关系、记录、索引和批次账零变化"]
    C -->|"是"| D["特征体系数据操作形成只含稳定句柄、强类型原始载荷和预期版本的不可变规格"]
    D --> E["事务外预读主键、槽位模板关系、唯一当前值关系、特征值类型化记录和 4170 批次账"]
    E --> F{"预读分类"}
    F -->|"同键同义且全部权威结构互证"| LR1["逻辑内返回：幂等读回原节点、关系、类型化记录和批次结果，零写入"]
    F -->|"同键异义、类型不符、旧版本或关系前置不满足"| LR2["逻辑内返回：冲突或版本漂移，零写入"]
    F -->|"许可竞争"| LR3["逻辑内返回：许可拒绝，可由调用方按协议重试"]
    F -->|"未找到且前置满足"| G["取得覆盖节点、关系、索引、特征值记录和批次账的同一唯一写入权"]
    G --> H["第一笔写入前锁内重读幂等账、主键、槽位、当前关系、旧值记录和来源版本"]
    H --> I{"锁内结果仍允许写入"}
    I -->|"否"| LR4["逻辑内收口：同义读回、异义冲突、版本漂移或许可拒绝；不形成候选"]
    I -->|"是"| J["按 NT-P1 接口创建带稳定主键的特征值节点候选；节点不绑定通用附属记录"]
    J --> K["初始槽位时创建关系 22 角色 1 宿主槽和角色 2 定义模板；换代时复核并失效旧角色 3 当前值关系"]
    K --> L["创建关系 22 角色 3 实例槽到新特征值的唯一当前值关系，并按需创建新值到合法来源的因果来源关系"]
    L --> M["向特征值类型化记录参与者登记一条本域记录：所属值节点、格式版本、记录状态、原始值版本和 I64 / VecI64 / VecU64 三者之一"]
    M --> N{"是否为具名特征批次"}
    N -->|"是"| N1["另向 4170 不可变批次账参与者登记完整有序项目、写前证据和发布后引用；不复制原始值"]
    N -->|"否"| O["继续统一准备"]
    N1 --> O
    O --> P["事务内完整读回：节点直接身份、槽位模板、唯一当前关系、来源关系、类型化记录和值版本"]
    P --> Q{"全部候选、参与者和请求语义是否一致"}
    Q -->|"否"| IR0["内部逻辑错误：停止新增写入，按批次账、值记录、关系、索引、节点的依赖逆序精确撤销"]
    IR0 --> IR1{"撤销后是否证明恢复事务前态"}
    IR1 -->|"否"| IR2["追根因并隔离事务域；不得释放后让普通读取观察半结构"]
    IR1 -->|"是"| IR3["返回内部不一致证据；不得改成普通拒绝或学习缺口"]
    Q -->|"是"| R["结构会话、特征值记录参与者和批次账参与者依登记顺序确认待发布"]
    R --> S{"任一确认失败"}
    S -->|"是"| IR0
    S -->|"否"| T["进入无失败最后发布区，关闭全部撤销能力并释放唯一写入权"]
    T --> U["普通读取按当前关系找到值节点，再读取该节点唯一当前类型化记录"]
    U --> V["复核节点稳定身份、记录所属节点、格式版本、原始值版本、三种载荷互斥和关系端点"]
    V --> W{"权威读回是否完整"}
    W -->|"否"| IR4["发布后内部逻辑错误：停止依赖路径，不重开候选撤销、不用缓存或旧主信息补值"]
    W -->|"是"| X["组合不可变特征值读取投影"]
    X --> X1["节点直接特征服务提供值式复核特征值归属入口；P2B 不得读取记录侧表"]
    X --> X2["节点直接特征服务提供值式读取宿主实例特征槽位组入口；P2C 不得直接读取关系或记录容器"]
    X --> Y["批次历史重放另读 4170 记录并按新值句柄、关系审计和类型化记录逐项互证"]
    Y --> Z["完成：I64、VecI64、VecU64 只有一个本域权威记录；关系独占拓扑；批次账独立"]
