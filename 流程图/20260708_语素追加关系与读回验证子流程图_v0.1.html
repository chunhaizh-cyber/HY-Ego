2.5 语素追加关系与读回验证子流程图

更新时间：2026-07-08

flowchart TD
    A["输入：语素入口 / 目标节点 / 关系类型"] --> B["检查语素入口是否有效"]
    B --> C{"入口是否为语素节点"}
    C -- "否" --> R1["拒绝：返回 false 或空集合"]
    R1 --> X["结构不变化"]
    C -- "是" --> D["2.2 目标节点准入"]
    D --> E{"目标节点是否可绑定"}
    E -- "否" --> R2["拒绝：返回 false"]
    R2 --> X
    E -- "是" --> F{"关系是否已存在"}
    F -- "是" --> R3["重复拒绝：返回 false，不重复写关系"]
    R3 --> X
    F -- "否" --> G{"追加类型"}
    G -- "对应信息" --> H["创建语素对应信息关系"]
    G -- "概念追溯" --> I["创建语素概念追溯关系"]
    H --> J["读取入口绑定目标组"]
    I --> K["读取入口概念追溯组"]
    J --> L{"读回是否包含目标"}
    K --> L
    L -- "否" --> R4["逻辑错误：写入后读回不符合预期"]
    R4 --> R5["追根因：关系仓库 / 调用参数 / 入口句柄"]
    L -- "是" --> S["输出只读材料：目标组 / 概念追溯组"]
