SQL Server 结构审计投影与控制面板查看流程图 v0.1

数据库只承载结构统计审计投影；运行期机器事实仍以内存仓库为准。

flowchart TD
    A["程序核心自检通过"] --> B["统计服务读取当前结构统计快照"]
    B --> C{"快照命名空间和结构计数可用?"}
    C -- "否" --> C1["逻辑内返回: 不发起数据库写入"]
    C -- "是" --> D["SQL 数据库适配验证服务器和数据库配置"]
    D --> E{"本地 SQL Server 可连接?"}
    E -- "否" --> E1["逻辑内返回: 形成数据库不可用材料并保持内存事实不变"]
    E -- "是" --> F["连接 master 并确保海中鱼巣数据库存在"]
    F --> G["连接业务数据库并确保结构统计审计表存在"]
    G --> H["参数化追加结构统计审计记录"]
    H --> I{"写入并读回是否符合预期?"}
    I -- "否" --> I1["追根因解决: 返回失败阶段并保持内存事实不变"]
    I -- "是" --> J["数据库审计写入完成"]
    J --> K["控制面板通过适配器查询最近审计记录"]
    E1 --> K
    K --> L{"查询是否成功?"}
    L -- "否" --> L1["逻辑内返回: 面板显示数据库不可用原因"]
    L -- "是" --> M["列表展示编号 时间 来源 节点 关系 索引"]
    M --> N["显示材料不写核心仓库且不裁决机器事实"]
