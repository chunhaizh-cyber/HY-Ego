## 说明

本图是显示层只读候选的代码逻辑边界图。当前没有独立显示层只读视图服务文件；当前控制台 `std::cout` 自检输出只能作为人读诊断，不能当作显示层只读视图能力完成。

## 流程图

```mermaid
flowchart TD
    A["显示请求：人读查看 / 调试观察 / 后续只读视图候选"] --> B["只读读取领域服务材料"]
    B --> C["可读材料：需求 / 任务 / 方法 / 状态 / 动态 / 因果 / 统计快照 / 日志审计材料"]
    C --> D{"读取是否走服务只读入口"}
    D -->|否| R0["拒绝：显示层不得绕过领域服务读取或写仓库"]
    D -->|是| E["组装只读展示材料"]
    E --> F{"展示材料是否试图承载机器逻辑"}
    F -->|是| R1["拒绝：标题 / JSON / HTML / 控制台文本不得裁决业务状态"]
    F -->|否| G["输出到控制台或后续只读视图"]
    G --> H{"显示层是否试图写业务事实"}
    H -->|是| R2["拒绝：不得写节点 / 主信息 / 关系 / 索引 / 需求 / 任务 / 方法"]
    H -->|否| I["显示完成；权威结构不变"]
    I --> J{"是否请求 WebView / DTO / JSON / 截图 / 控制面板"}
    J -->|当前未立项| R3["保持候选；不接入新显示通道"]
    J -->|后续专项确认| K["另建只读显示详细设计和施工计划"]
```

## 关键边界

```text
显示层只读，不写机器事实。
控制台输出、日志、HTML、JSON、截图和显示标题只做人读材料。
显示层不得绕过领域服务写节点仓库、主信息仓库、关系仓库或索引仓库。
当前没有 DTO / JSON / WebView / 窗口 / 截图 / 控制面板接入口。
```

## 当前代码差距

```text
当前没有独立显示层只读视图服务。
当前只有控制台自检输出，不能声明显示层只读视图完成。
当前没有跨服务只读 DTO、稳定显示契约或显示层验收入口。
```
