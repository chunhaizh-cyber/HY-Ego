# 2.3 语素入口创建子流程图

更新时间：2026-07-08

## 依据

```text
海中鱼巣/领域/语素服务.h
海中鱼巣/入口.cpp
```

## 说明

本子流程表达 `语素服务::创建语素入口` 的代码逻辑。它只在对应信息节点准入通过后写结构。

## 流程图

```mermaid
flowchart TD
    A["输入：对应信息节点 / 可选主键"] --> B["2.2 对应信息节点准入"]
    B --> C{"准入是否通过"}
    C -- "否" --> R["返回无效句柄"]
    R --> R1["结构不变化：不创建主信息 / 语素节点 / 关系 / 索引"]
    C -- "是" --> D["主信息仓库创建主信息"]
    D --> E["节点仓库创建语素节点"]
    E --> F["关系仓库创建语素对应信息关系"]
    F --> G{"主键是否非 0"}
    G -- "是" --> H["索引仓库绑定主键到语素节点"]
    G -- "否" --> I["跳过索引绑定"]
    H --> S["返回语素入口句柄"]
    I --> S
```

## 关键边界

```text
创建入口前必须先排除不可绑定节点。
语素入口写入结构为主信息、语素节点、语素对应信息关系和可选主键索引。
本流程不解析文本、不创建概念追溯、不自动创建对应信息节点。
```
